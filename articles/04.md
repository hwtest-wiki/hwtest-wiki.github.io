# 工程师的工具箱：万用表 / 电源 / 电子负载 / 信号源，到底怎么用

<div class="article-meta"><span class="badge">📘 模块一 · 测试基础与方法论</span><span class="badge">⏱ 约 13 分钟</span></div>


示波器很风光，但真到了台面上干活，你会发现 80% 的时间，手其实是放在另外四样东西上的：

- **万用表**——量一个电压、看通不通；
- **直流电源**——给电路供电；
- **电子负载**——测电源带不带得动；
- **信号源**——给电路喂一个输入信号。

它们看着都「简单」，可恰恰是这种「我会用」的错觉，最容易出事——**接线接错一根、档位选错一挡，测出来的数据就是错的，你还浑然不觉。** 这一篇，就把这四件套的「正确打开方式」和**最常踩的坑**一次说清。

**读完这篇，你能搞懂：**

- ✅ 万用表测电压、测电流为什么**接法完全相反**，接反了会怎样；
- ✅ 直流电源的 **CV / CC 两种状态**，以及「限流」这个救命设置；
- ✅ 为什么精密测量要用**四根线（开尔文接法）**，两根线差在哪；
- ✅ 电子负载到底是个**什么东西**，工程师拿它来「烤」什么；
- ✅ 信号源的**幅度为什么常会差一倍**——50Ω 阻抗匹配的坑。

## 一、万用表：最熟，也最容易想当然

万用表（DMM）几乎是每个人接触的**第一件**电子仪器。正因为太熟，反而最容易栽跟头。先看它到底能测什么：

- **电压（V）**：最常用，量某两点之间「电势差」是多少伏；
- **电流（A）**：电路里流过多大的电流；
- **电阻（Ω）**：一个电阻多大、或者元件有没有开路/短路；
- **通断（🔊）**：两点之间通不通，通了「嘀」一声，查断线神器；
- **二极管 / 电容 / 频率**：进阶档位，看具体型号支持。

![图2：万用表能测的五种量](/images/04/ill02_dmm_measures.png)

这里要先讲一个**很多人一辈子没搞清**、却天天在用的概念——**真有效值（True RMS）**。

> 💡 **什么是「真有效值」？** 交流电压时高时低、来回变，没法用「某一瞬间的值」来代表它。于是工程师用**有效值**来描述：一个交流电压的有效值是 5V，意思是它**发热的本事，和一个 5V 的直流电压一样**。
>
> 问题来了：**便宜的万用表**测交流，是偷懒按「标准正弦波」去**估算**的；可现实里的信号（比如开关电源、调光电路）波形歪七扭八，这一估就**错得离谱**。而标着 **True RMS（真有效值）** 的表，是老老实实**按真实波形算**的，再奇怪的波形都测得准。**测非正弦的交流，认准 True RMS。**

### 测电压和测电流，接法是「反」的

这是万用表**头号大坑**，新手几乎都中过招。记住一句话：**测电压并联，测电流串联。**

- **测电压**：表笔**搭在**被测两点上（**并联**），就像拿尺子贴着量长度，不打断电路；
- **测电流**：得把电路**断开**，让电流**穿过**万用表（**串联**），就像在水管中间装一个流量计，水必须从它身体里流过去。

![图3：测电压「并联」vs 测电流「串联」——接法完全相反](/images/04/ill03_dmm_connect.png)

为什么必须这样？这背后是万用表的**内阻**在起作用：

> 🔧 **想当然就翻车**：
> - **测电压**时，万用表内阻**极大**（兆欧级），几乎不分走电流，所以并联上去不影响电路；
> - **测电流**时，万用表内阻**极小**（接近一根导线），串进去才不影响电流大小。
>
> **最经典的炸表事故**：表还停在「电流挡」（内阻极小），你却直接拿它去**并联量电源电压**——相当于拿一根导线把电源正负极短路！轻则烧保险丝，重则打火、报废表。**用完电流挡，记得立刻拨回电压挡**，这是肌肉记忆级的安全习惯。

## 二、直流电源：会「自己变脸」的供电器

直流电源（DC Power Supply）给电路供电，看似就是「调个电压、按下输出」。但它其实有**两种工作状态**，会根据负载**自动切换**——搞不懂这个，你会被它「忽悠」。

### CV 与 CC：恒压和恒流

- **CV（恒压，Constant Voltage）**：你设 5V，它就死死咬住 5V 输出，电流多少由负载决定。**这是平时供电的正常状态。**
- **CC（恒流，Constant Current）**：当电流大到撞上你设的**限流值**，它会**牺牲电压**、转而死死咬住电流不再往上涨。

![图4：直流电源的 CV / CC 两种状态，会自动切换](/images/04/ill04_psu_cvcc.png)

这个切换是**自动**的，规律很简单：**谁先撞到上限，就由谁说了算。**

> 💡 **打个比方**：CV 像「定速巡航」——咬住速度（电压）不放；可一旦上坡太陡、油门（电流）踩到底也提不上速，就只能保住油门最大值（CC），让速度掉下来。电源也是：电压顶不住了，就退守恒流。

### 「限流」不是麻烦，是保命

很多新手嫌限流（电流上限）设置麻烦，直接拉到最大。**这是拿板子冒险。**

> 🔧 **限流 = 你的安全气囊**：假设你的板子正常只吃 200mA，那就把限流设在 **300mA 左右**。万一板子上有个元件焊反了、或哪里短路了，电流会瞬间冲很高——这时电源**立刻进入 CC 状态，把电流摁在 300mA**，电压自动垮下来。板子最多有点热，**不会被大电流烧穿**。
>
> 如果限流拉满（比如 5A），同样一个短路，5A 电流灌进去，**冒烟、烧线、报废芯片**就在一瞬间。**养成习惯：上电前，先按板子的正常电流，把限流设好。**

### 四线「开尔文」接法：精密测量的必修课

当你要给电路供一个**很准**的电压，或者电流比较大时，会发现一个怪事：**电源表头显示 5.00V，可板子上实测只有 4.85V。** 电压去哪了？

**答案：消耗在那两根供电导线自己的电阻上了。** 导线再粗也有电阻，电流一大，「欧姆定律 V = I×R」就在线上**白白吃掉**一截电压（这叫**线损**）。

解决办法，就是大名鼎鼎的**四线（开尔文，Kelvin）接法**：

![图5：限流保护 + 四线开尔文接法——把线损「踢出」测量](/images/04/ill05_psu_kelvin.png)

> 💡 **四线为什么准？** 两根**粗线**专门**送电流**（force），另两根**细线**专门**在负载端「就地」测电压**（sense）。这对 sense 线几乎不流电流，所以**它们自己不产生线损**；电源拿这个「就地实测」的真实电压去**反过来补偿**——线上掉了多少，它就在源头多补多少，保证**负载端**正好是你要的 5V。
>
> 一句话：**force 线负责出力，sense 线负责说真话**，把导线电阻彻底「踢出」测量回路。需要精密供电时，四线接法是基本功。

## 三、电子负载：一个专门「假装耗电」的设备

电子负载（DC Electronic Load）是四件套里**新手最陌生**的一个。它长得像台仪器，作用却很「奇怪」——**它专门用来「耗电」。**

你可以把它理解成一个**可以随意调节大小的「电老虎」**：想让它吃 1A 就吃 1A，想让它吃 10A 就吃 10A。那要它何用？——**用来考验电源。**

> 💡 **为什么需要它？** 你设计了一个充电器、一个 DC-DC 模块，标称能输出「5V / 3A」。可它**真的**能在 3A 满载时稳稳输出 5V 吗？带载到 3A 时电压会不会垮？会不会发烫、保护性关机？你需要一个「**假负载**」来逼它满负荷干活——这就是电子负载。它比一堆大电阻灵活得多：**电流大小可以随手一拧就变，还能动态突变**，模拟真实设备一会儿轻载、一会儿重载。

它有**四种工作模式**，对应四种「考法」：

![图6：电子负载的四种模式——CC / CV / CR / CP](/images/04/ill06_load_modes.png)

- **CC（恒流）**：最常用。固定吃 3A，看电源在 3A 下电压稳不稳——**测带载能力**；
- **CV（恒压）**：把电压钳在某值，常用来**测充电器/电池**的行为；
- **CR（恒阻）**：模拟一个固定阻值的电阻，电流随电压线性变；
- **CP（恒功率）**：固定吃 30W，电压高就少吃点电流、电压低就多吃点——**模拟很多真实电子设备的「吃电」特性**。

下面这张图，就是工程师最常做的一件事——**用电子负载「烤」一个电源**：

![图7：用电子负载「烤」电源——拉满电流，看输出稳不稳](/images/04/ill07_load_test.png)

> 🔧 **实战场景**：把待测电源的输出，接到电子负载上，负载设成 **CC 模式**，电流从 0 一点点往上加到满载 3A。同时用万用表（或示波器）盯着电源输出电压：
> - 电压**稳稳不动** → 这个电源带载能力合格；
> - 电流加大、电压就往下**掉** → 带载能力差，或者线/接触有问题；
> - 加到某个电流**突然关断** → 它的**过流保护**触发了，正好测出保护点在哪。
>
> 这一步，是几乎所有电源测试的「**规定动作**」。

## 四、信号源：给电路「喂」一个输入

前面三件，要么测、要么供电。**信号源（信号发生器）**则相反——它负责**主动产生**一个信号，**喂给**你的电路当输入，再用示波器看电路「消化」得对不对。

它能产生的，通常是这几样：正弦波、方波、三角波，外加你能调的三个核心参数：

- **频率**：信号一秒钟来回多少次（Hz）；
- **幅度**：信号有多「高」，峰峰值多少伏（Vpp）；
- **偏置（Offset）**：让整个波形整体**上移或下移**，比如让一个交流波形「坐」在 1.5V 的直流台子上。

![图8：信号源的波形 / 频率 / 幅度 / 偏置，以及 50Ω 的坑](/images/04/ill08_siggen.png)

讲到信号源，有一个**几乎人人都被坑过**、却很少有人讲明白的陷阱——**幅度莫名其妙差了一倍。**

> 🔧 **「我明明设了 1Vpp，示波器却显示 2Vpp」**：这不是仪器坏了，是**阻抗匹配**在作怪。信号源的输出端，内部有一个 **50Ω 的内阻**。它出厂默认**假设你接的负载也是 50Ω**——这时信号源内阻和负载**各分掉一半电压**，所以它会**偷偷把内部电压调到 2 倍**，好让你在 50Ω 负载上**正好**得到设定的 1Vpp。
>
> 可示波器的输入阻抗通常是 **1MΩ（高阻）**，几乎不分走电压。于是信号源「预补偿」的那 2 倍，**原封不动全到了示波器上**——你就看到了 2Vpp。
>
> **怎么破？** 在信号源菜单里把**输出负载设置**从「50Ω」改成「**高阻（High-Z）**」，告诉它「我接的是高阻设备，别自作主张补偿」，显示就对上了。**接示波器/高阻电路，记得把负载设成 High-Z。**

> 💡 那 50Ω 又是哪来的、什么时候真要匹配？简单说：在**高频**世界里，线缆会像「水管」一样反射信号，为了不反射，信号源、线缆、负载三者阻抗都要做成 **50Ω**（这是射频领域的通用标准）。低速实验里接高阻没问题，但一旦上了高频，**50Ω 匹配**就是另一门必修课了。

## 五、四件套怎么配合？一个真实例子

单独认识完，看它们**合伙干活**才有体感。假设你要测一个 **DC-DC 降压模块**（输入 12V，输出 5V/3A），完整一套会这么摆：

1. **直流电源** → 给模块喂 12V 输入，**限流先设好**（比如 1A），上电；
2. **万用表** → 量一下输出，确认是不是 5V，先看个大概；
3. **电子负载** → 接到 5V 输出上，CC 模式从 0 拉到 3A，**逼它满载**；
4. **示波器 + 探头** → 同时盯着输出，看满载时**电压跌多少、纹波大不大**（纹波怎么测，正是后面要专门讲的硬骨头）。

你看，**供电的、测量的、加载的、看波形的**，四件套各司其职，一台完整的「电源体检」就跑起来了。**工具不在多，在于每一件都用对。**

## 六、一张速查表，上手前对一遍

![图9：工程师工具箱 · 使用速查卡](/images/04/ill09_cheatsheet.png)

| 仪器 | 关键用法 | 最容易踩的坑 |
|---|---|---|
| **万用表** | 测压**并联**、测流**串联** | 电流挡没拨回，去量电压 → **炸表** |
| **万用表** | 测交流认准 **True RMS** | 普通表测非正弦波，**数值全错** |
| **直流电源** | 上电先设好**限流** | 限流拉满，短路直接**烧板** |
| **直流电源** | 精密供电用**四线开尔文** | 两线接法，线损吃掉一截电压 |
| **电子负载** | CC 模式**拉满电流**烤电源 | 忘了散热/超功率，**烧负载** |
| **信号源** | 接示波器设 **High-Z** | 默认 50Ω，幅度**差一倍** |

## 写在最后

示波器和探头是工程师的「**眼睛**」，那这四件套，就是工程师的「**双手**」：

- **直流电源**给电路**供血**；
- **万用表**随手**把脉**；
- **电子负载**专门**出难题、压测极限**；
- **信号源**负责**递话筒、给输入**。

它们没有示波器那么「高科技」，却是你每天最先碰、用得最多的家伙。**把这四件的接法和那几个坑刻进肌肉记忆，你的测量地基就稳了**——后面再难的测试，都是在这个地基上盖楼。

> **工具箱的第一准则：越「简单」的仪器，越要敬畏。数据错，往往不是仪器不准，而是你用错了它。**

下一篇，我们给这套工具箱来一次「灵魂拷问」——仪器都会用了，可它读出来的数，就一定是真的吗？**测量为什么会不准？我们把误差、精度、校准和单位一次讲透，让你看懂表上那串数字，到底有几位能信。**
