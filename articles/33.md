# S 参数：低频量插入损耗才 1dB、挺好，为什么高速一跑眼图还是塌了？——不扫全频段，你根本看不见信号是怎么被吃掉的

<div class="article-meta"><span class="badge">📘 模块三 · 信号完整性 SI · S 参数 / 插入损耗 / 回波损耗</span><span class="badge">⏱ 约 18 分钟</span><span class="badge qr">🔧 含工程师速查</span></div>


一条高速链路，板子回来上测试台。工程师想先确认走线本身干不干净，就用仪器量了一下**插入损耗**——信号从一头进、另一头出，损了多少。他在 1GHz 上测了一个点：**才 1dB**，几乎没损。他很满意：走线挺好。

可真跑起高速信号，眼图还是塌了、误码率还是不达标。他把损耗**从直流一路扫到 10GHz**，这才傻眼：损耗**不是一个数，是一条随频率往下掉的曲线**——1GHz 才 1dB，到 4GHz 掉到 6dB，到 8GHz 已经掉到 **15dB**，也就是信号功率在这个频率上**被走线吃掉了 97%**。而他的信号，能量恰恰主要落在这个高频段。

**问题就出在这：他只看了 1GHz 这一个低频点，却没看损耗在高频有多凶。** 高速走线的损耗**随频率单调增大**——频率越高，损得越狠。你在低频量一个点说"挺好"，到了信号真正用的高频段，可能早就塌到没法看了。**单点、低频会骗你，你必须把整个频段扫一遍。**

**而把反射、损耗、串扰按频率一次全量清楚的那张表，就是 S 参数。** 这一篇，我们就从"S 参数到底是什么"讲起，把 S11、S21 怎么读、怎么测、怎么和一致性模板对，一条链讲透。

**读完这篇，你能搞懂：**

- ✅ S 参数到底是什么——把链路当**黑盒网络**，用**进去的波和出来的波之比**描述它；
- ✅ S11 回波损耗——**反射了多少**，怎么和阻抗匹配、TDR 对上；
- ✅ S21 插入损耗——**透过去多少**、损耗为什么随频率涨、**3dB 带宽**怎么看；
- ✅ dB 怎么读——**3dB=功率剩一半**、**20dB=反射只剩 1%**，一眼换算；
- ✅ 怎么测——**矢量网络分析仪（VNA）扫频 + 校准 / 去嵌入**，加**TDR** 时域互补定位；
- ✅ 怎么和**一致性测试**对——插损 / 回损 / 串扰要落在标准的模板线以内。

## 一、S 参数是什么：把链路当黑盒，量"出去的波比进来的波"

先说一个高速里绕不开的观念转变。低频、直流的时候，我们习惯用**电压、电流、电阻**描述一段线。可到了高频，一段走线的长度已经能和信号波长比了，**电压、电流沿着线一路都在变**，"这根线电阻多少欧"这种说法就不够用了。高频下真正的主角是**阻抗**和**波的传播**。

所以工程师换了个思路：把一段走线、一个连接器、一整条链路，都当成一个**黑盒网络**，不管它内部长啥样，只在它的**端口**上做一件事——**往里打一个已知的波（入射波），量从各个端口出来的波（反射波、透射波），拿出来的波比进去的波，就得到这个网络的一组"传递系数"。这组系数，就是 S 参数（Scattering Parameters，散射参数）。**

拿最常见的**二端口网络**（比如一段走线，左边端口 1、右边端口 2）来说，从端口 1 打进去一个波，会发生两件事：

- 一部分**原路反射**回端口 1——它和入射波的比值，就是 **S11**；
- 一部分**透过去**从端口 2 出来——它和入射波的比值，就是 **S21**。

反过来从端口 2 打进去，就有 **S22（端口 2 的反射）** 和 **S12（从 2 透到 1）**。所以二端口有四个 S 参数，凑成一个 2×2 的矩阵。**记法很简单：Sij = 端口 j 激励、从端口 i 出来的响应。头一个下标是"出口"，第二个下标是"入口"。** 下标相同（S11、S22）是**反射**，下标不同（S21、S12）是**透射**。

![图1：翻车现场——高速链路只在1GHz测一个插入损耗点，才1dB看着挺好；可把损耗从DC扫到10GHz，S21是一条随频率往下掉的曲线：1GHz=-1dB、4GHz=-6dB、8GHz=-15dB（功率被吃掉97%），信号能量恰落在高频段，眼图照样塌。只看低频单点会骗你，必须扫全频段](/images/33/ill01_scene.png)

> **一句话记住：S 参数把链路当黑盒，只量"出来的波 ÷ 进去的波"。Sij 里第一个下标是出口、第二个是入口；下标相同=反射（S11/S22），下标不同=透射（S21/S12）。它天生就是按频率一根根量的——单点、低频看不出真相。**

## 二、S 矩阵与四个参数：反射看对角线，透射看非对角

把二端口的四个 S 参数摆成矩阵，一眼就能分工：

**对角线上的 S11、S22 管"反射"**，也就是**回波损耗**——从这个端口打进去，有多少被原路弹回来。反射越小越好，说明**阻抗越匹配**。

**非对角线上的 S21、S12 管"透射"**，也就是**插入损耗**——信号从一端传到另一端，透过去了多少、损了多少。透射越接近 0dB（无损）越好。对大多数无源走线，S21 和 S12 是相等的（互易网络）。

**S 参数不止用于二端口。** 一对差分线、几条挨着的走线，就是**多端口网络**——串扰，本质上就是**从"打扰者（aggressor）"端口进、从"受害者（victim）"端口出**的那个透射 S 参数：同一侧出来的叫**近端串扰 NEXT**，另一侧出来的叫**远端串扰 FEXT**。所以**反射、损耗、串扰，在 S 参数体系里是统一的一套语言**，全都是"某端口进、某端口出"的比值，全都随频率变化。

**为什么 S 参数都用 dB、都是频率的函数？** 因为高频器件的行为**强烈依赖频率**：同一条走线，低频几乎不损，高频损得厉害；同一个连接器，某个频率突然谐振、反射暴涨。**只报一个数没有意义，必须给一条"S 参数 vs 频率"的曲线。** 而幅度动辄差几个数量级，用线性比值不好看，于是统一取 dB。

![图3：S参数矩阵。二端口2×2矩阵：对角S11/S22=反射=回波损耗（阻抗匹配好则小），非对角S21/S12=透射=插入损耗（越接近0dB越好，无源互易网络S21=S12）。多端口时串扰=aggressor端口进、victim端口出的透射S参数，近端NEXT/远端FEXT。反射损耗串扰统一成"某端口进某端口出"的比值，都是频率的函数、都用dB](/images/33/ill03_matrix.png)

## 三、dB 怎么读：3dB 功率剩一半，20dB 反射只剩百分之一

S 参数全是 dB，先把 dB 的读法定死，后面看曲线才不慌。

**dB 是个比值的对数刻度。** 对**功率**：dB = 10 × log₁₀(功率比)；对**电压**：dB = 20 × log₁₀(电压比)。记住几个关键锚点，其它靠内插：

| dB 值 | 功率比 | 电压比 | 直觉 |
|---|---|---|---|
| **0dB** | 1（100%） | 1 | 全透 / 全反，没损 |
| **−3dB** | 0.5（50%） | 0.71 | **功率剩一半**（带宽就定在这） |
| **−6dB** | 0.25（25%） | 0.5 | 电压剩一半 |
| **−10dB** | 0.1（10%） | 0.32 | 剩十分之一 |
| **−20dB** | 0.01（1%） | 0.1 | **只剩百分之一** |
| **−40dB** | 0.0001 | 0.01 | 万分之一，基本没了 |

**读 S21（插入损耗）时**：0dB 是理想全透，越负损得越多。**−3dB 是个分水岭——信号功率损掉一半**，所以**"带宽"通常就定义在 S21 掉到 −3dB 的那个频率**。−10dB 就只剩十分之一功率，链路基本废了。

**读 S11（回波损耗）时**：越负越好（反射越小）。工程上常把 S11 取正号叫"回波损耗（RL）"，比如"回损 20dB"就是 S11 = −20dB，意思是**反射功率只有入射的 1%**——匹配相当好。**−10dB（反射 10%）算勉强，−15 到 −20dB 是常见的合格线，越负越干净。**

> **一句话记住：dB = 20log(电压比) = 10log(功率比)。−3dB 功率剩一半（带宽点），−10dB 剩十分之一，−20dB 只剩百分之一。S21 看损耗越接近 0 越好、掉到 −3dB 定带宽；S11 看反射越负越好、−20dB=反射仅 1%。**

## 四、S11 回波损耗：反射多少，就是阻抗匹配得好不好

S11 量的是**从端口 1 打进去、又被原路弹回来的比例**，它直接反映**阻抗匹配**。

回想第 28 篇：信号在走线上遇到**阻抗不连续**（接头、过孔、拐角、宽度突变），就会在那里**反射**。反射越多，S11 越大（越接近 0dB）；阻抗越连续、越接近特征阻抗 Z₀（一般 50Ω 单端、100Ω 差分），反射越小，S11 越负。

**S11 和阻抗的关系有个直接公式。** 反射系数 Γ =（Z − Z₀）/（Z + Z₀），而 **S11(dB) = 20 × log₁₀|Γ|**。举两个例子（Z₀=50Ω）：

- 某处阻抗 60Ω → Γ =（60−50）/（60+50）≈ 0.091 → S11 ≈ **−21dB**，反射不到 1%，很好；
- 某处阻抗 75Ω → Γ =（75−50）/（75+50）= 0.2 → S11 ≈ **−14dB**，反射 4%，勉强；差得越远，反射越凶。

**S11 曲线还会告诉你"链路上哪里有问题"。** 如果 S11 在某个频率突然拱起一个包，往往是某个不连续点在那个频率发生了谐振——比如一段过孔的 **stub（残桩）**、一个阻抗突变。

**但 S11 是频域的，它把整条线的反射混在一起，说不清"反射发生在离端口多远的地方"。** 这时候就要请出它的**时域搭档 TDR**（时域反射计）：往线里打一个快边沿的阶跃，沿途每遇到一个阻抗不连续就弹回一点，示波器按**回来的时间**就能算出**不连续点离端口多远**，按**弹回的幅度**算出**那里阻抗是高了还是低了**。**TDR（时域看位置）和 S11（频域看反射量）是一对傅里叶变换关系——一个告诉你"哪里坏"，一个告诉你"坏多少、在哪个频率坏"，配合起来定位最快。**

![图4：S11回波损耗。上：S11曲线越负越好，−20dB=反射仅1%（合格），某频率拱包=不连续点谐振（如过孔stub）。中：反射系数Γ=(Z−Z0)/(Z+Z0)，S11=20log|Γ|；60Ω→−21dB好、75Ω→−14dB勉强。下：TDR时域搭档，打快边沿阶跃，按回波时间定位不连续点离端口多远、按幅度判阻抗高低；TDR与S11是傅里叶变换对，一个看位置一个看频率](/images/33/ill04_s11.png)

## 五、S21 插入损耗：透过去多少，损耗为什么越到高频越凶

S21 量的是**信号从端口 1 透到端口 2、还剩多少**，它反映链路的**插入损耗**。理想走线 S21=0dB（全透），实际总要损一点，频率越高损得越多。

**为什么损耗随频率单调增大？** 两个主要原因：

- **导体损耗（趋肤效应）：** 频率越高，电流越往导体表面挤，等效截面越小、电阻越大。这部分损耗大致**正比于频率的平方根（√f）**。
- **介质损耗：** PCB 板材（FR4 等）在交变电场下会发热吸能，频率越高吸得越多。这部分损耗大致**正比于频率（f）**。

两者叠加，就是那条**越到高频掉得越狠**的 S21 曲线。所以开头那条链路，1GHz 才 −1dB，8GHz 掉到 −15dB，一点都不奇怪。

**S21 曲线上要盯两样东西：**

1. **整体下滑趋势 + 3dB 带宽：** S21 掉到 −3dB 的频率，就是链路的**带宽**。信号的主要能量频率如果超过这个带宽，边沿就会被磨钝、眼图就会塌——这正好接上第 29 篇"带宽不够边沿变缓"。
2. **突然的凹陷（谐振点）：** 如果 S21 在某个频率**猛地掉下去一个深坑**，通常是**过孔 stub、走线分支、或者周期性结构**在那个频率发生了谐振，把能量吸走或反射掉。这种深坑最要命——它可能正好砸在你信号的某个关键频率上。

**串扰也用 S21 这类透射参数看：** 把干扰线当激励、被扰线当输出，量出来的 NEXT / FEXT 曲线越低（越负）越好，一致性测试同样会给串扰的模板线。

![图5：S21插入损耗。上：S21随频率单调下滑曲线，掉到−3dB处定"带宽"，信号主能量超带宽则边沿钝、眼图塌（接第29篇）。中：损耗两来源——导体趋肤效应∝√f、介质损耗∝f，叠加成高频掉得更狠。下：曲线上的突然深坑=谐振点（过孔stub/分支/周期结构），可能砸中信号关键频率；串扰NEXT/FEXT也是透射S参数、越低越好](/images/33/ill05_s21.png)

## 六、怎么测：矢量网络分析仪扫频，校准是灵魂

量 S 参数的主力仪器是**矢量网络分析仪（VNA，Vector Network Analyzer）**。它的工作方式很直白：**内部信号源从起始频率到终止频率一点点扫过去，每个频率都往被测件（DUT）打一个已知的波，同时用接收机量出反射波和透射波的幅度和相位，一除，就得到那个频率的 S11、S21，扫完整个频段，就画出整条 S 参数曲线。**（"矢量"就是指它连相位一起测，不只是幅度。）

**但 VNA 测 S 参数，校准（calibration）才是灵魂。** 因为你接的线缆、接头、测试夹具本身就有损耗和反射，不校准的话，测出来的是**"线缆 + 夹具 + DUT"的总账**，冤枉了 DUT。校准要做的，就是**把测量的参考面从仪器端口"推"到 DUT 的真正入口**，把线缆夹具的影响扣掉：

- **SOLT 校准：** 依次接**短路（Short）、开路（Open）、负载（Load）、直通（Thru）**四个已知标准件，让 VNA 反推出误差模型，最常用；
- **TRL 校准：** 用**直通 / 反射 / 传输线**三种标准，适合板上、夹具环境，精度高；
- **去嵌入（de-embedding）：** 如果 DUT 前面还有一段没法直接校掉的夹具，就先单独量出夹具的 S 参数，再从总测量里**数学上剥掉**，得到 DUT 净值。

**别忘了时域搭档 TDR。** VNA 在频域看"哪个频率损/反射多少"，TDR 在时域看"沿线哪个位置阻抗突变"。真定位问题时，两个一起用：VNA 的 S11 说"反射超标了"，TDR 就顺着线告诉你"是第 3 个过孔那儿阻抗掉了"。

**最后落到一致性测试。** USB、PCIe、以太网、HDMI 这些高速接口的规范里，都直接给了 S 参数的**模板线（limit line / mask）**：插入损耗 S21 不能低于某条线（损耗不能太大）、回波损耗 S11 不能高于某条线（反射不能太多）、串扰也有上限。**测出来的曲线只要有一段越过模板线，这一项就判不合格。** 这就是 S 参数在一致性测试里的硬核用途——它把"信号干不干净"变成了**一条条能自动判过 / 不过的客观曲线**。

![图6：怎么测S参数。VNA矢量网络分析仪从起始到终止频率扫频，每点打已知波、连幅度和相位一起测反射波透射波，一除得S11/S21。校准是灵魂：SOLT（短路/开路/负载/直通）或TRL把参考面从仪器端口推到DUT入口、扣掉线缆夹具；去嵌入数学剥掉夹具。TDR时域搭档定位不连续位置。一致性测试给S21/S11/串扰模板线（limit line），曲线越线即判不合格](/images/33/ill06_measure.png)

> **一句话记住：VNA 扫频测 S 参数，校准（SOLT/TRL）把参考面推到 DUT、扣掉线缆夹具，是测准的灵魂；去嵌入剥掉夹具。TDR 时域定位不连续点，和 S11 傅里叶互补。一致性测试用模板线自动判插损/回损/串扰过不过。**

## 七、怎么用起来：一条完整的排查链

把这一篇串成一个可操作的动作：

**① 先扫全频段，别只看一个点。** 用 VNA 从直流扫到信号最高有用频率的 3～5 倍，拿到 S11、S21 整条曲线。

**② 校准先做到位。** SOLT 或 TRL 把参考面推到 DUT，必要时去嵌入夹具——否则你测的是线缆，不是板子。

**③ 看 S21（损耗）：** 3dB 带宽够不够信号用？有没有突然的谐振深坑砸在关键频率上？损耗太大 → 缩短走线 / 换低损耗板材 / 走内层；有深坑 → 查过孔 stub、走线分支。

**④ 看 S11（反射）：** 有没有超过合格线（常见 −15～−20dB）？哪个频率拱包？配合 **TDR** 顺着线找到具体的阻抗不连续点（过孔、接头、拐角），回去改叠层 / 补偿。

**⑤ 看串扰（NEXT/FEXT）：** 超标就回到第 30 篇——拉间距、加护地、控好回流路径。

**⑥ 对模板线判过 / 不过。** 一致性测试直接拿标准的 limit line 卡，任何一段越线就是不合格，按上面分工回去治。

## 🔧 工程师速查（附录）

### 一、S 参数是什么

| 名词 | 含义 |
|---|---|
| S 参数（散射参数） | 把网络当黑盒，量"出来的波 ÷ 进去的波"的一组频率相关系数 |
| Sij 下标 | 端口 j 激励、端口 i 响应；第一下标=出口，第二下标=入口 |
| 反射 / 透射 | 下标相同（S11/S22）=反射；下标不同（S21/S12）=透射 |
| 为什么按频率 | 高频行为强依赖频率，必须给"S 参数 vs 频率"曲线，不能只报一个数 |

### 二、四个参数分工（二端口）

| 参数 | 名称 | 含义 | 好的方向 |
|---|---|---|---|
| **S11** | 回波损耗（输入反射） | 端口 1 打进去、原路弹回的比例 | 越负越好（反射小、匹配好） |
| **S21** | 插入损耗（正向透射） | 端口 1 → 端口 2 透过去的比例 | 越接近 0dB 越好（损耗小） |
| **S12** | 反向透射 | 端口 2 → 端口 1；无源互易网络 = S21 | 同 S21 |
| **S22** | 输出反射 | 端口 2 打进去、原路弹回的比例 | 越负越好 |
| NEXT/FEXT | 近端/远端串扰 | 干扰线激励、被扰线响应的透射 | 越负越好 |

### 三、dB 读法（背下来）

- **dB = 20·log₁₀(电压比) = 10·log₁₀(功率比)**
- 0dB=100% ｜ **−3dB=功率 50%**（带宽点）｜ −6dB=电压 50% ｜ −10dB=10% ｜ **−20dB=1%** ｜ −40dB=0.01%
- S21 越接近 0 越好，掉到 −3dB 定带宽；S11 越负越好，−20dB=反射仅 1%（合格常在 −15～−20dB）。

### 四、S11 与阻抗

- 反射系数 **Γ =（Z − Z₀）/（Z + Z₀）**，**S11(dB) = 20·log₁₀|Γ|**（Z₀ 常见 50Ω 单端 / 100Ω 差分）。
- 例：60Ω→S11≈−21dB（好）；75Ω→S11≈−14dB（勉强）。
- S11 某频率拱包 = 不连续点谐振（过孔 stub 等）。
- **TDR**（时域反射）与 S11 是傅里叶变换对：TDR 看不连续点**位置 + 阻抗高低**，S11 看**反射量 vs 频率**。

### 五、S21 与损耗

- 损耗随频率**单调增大**：导体趋肤效应 **∝√f**、介质损耗 **∝f**，叠加成高频更狠。
- **−3dB 带宽**：S21 掉到 −3dB 的频率；信号主能量超带宽 → 边沿钝、眼图塌。
- 突然**深坑** = 谐振（过孔 stub / 分支 / 周期结构），最怕砸中信号关键频率。
- 治损耗：缩短走线、低损耗板材、走内层；治深坑：查 stub / 分支。

### 六、怎么测

- **VNA（矢量网络分析仪）**：扫频，每点连幅度和相位测反射 / 透射，得整条 S 参数。
- **校准是灵魂**：SOLT（短路/开路/负载/直通）或 TRL（直通/反射/传输线）把参考面推到 DUT、扣线缆夹具；**去嵌入**数学剥掉夹具。
- **TDR** 时域定位不连续点，和 S11 互补。
- **一致性测试**：规范给 S21/S11/串扰**模板线（limit line）**，曲线越线即不合格。

### 七、常见坑

- ☐ **不校准 / 参考面没推到 DUT** → 测的是线缆夹具，冤枉板子。
- ☐ **只看低频或单点插损** → 高频段 S21 早塌了没发现。
- ☐ 把 **S11 和 S21 的好方向搞反**（S11 越负越好、S21 越接近 0 越好）。
- ☐ 忽略 **S21 深坑（谐振）** → 正好砸中信号关键频率、眼图莫名塌。
- ☐ 带宽 / 探头 / 地线老规矩不守 → 测量本身引入误差。
