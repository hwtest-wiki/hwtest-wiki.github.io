# 万用表测着 1.0V 稳如泰山，CPU 一满载，电压却砸出一个坑——那一下去哪了？

先讲个真事。

一块板子，主控芯片的供电是 **1.0V**。工程师拿**万用表**量、拿**示波器**看稳态波形——**1.0V 稳稳的，纹波、精度全在规格内，分毫不差**。可一跑大型游戏、一进加载界面，**芯片就莫名其妙重启**。

反复查供电，稳态测了一遍又一遍，**全都合格**。问题到底出在哪？

后来用**电子负载**模拟了一下"突然满载"的那一瞬间，示波器上才露了馅：**就在负载跳起来的那一刻，1.0V 的电压被瞬间"拽"下去，砸出一个坑，最低掉到 0.90V——而芯片的下限是 0.95V。** 它在那一下，就饿晕了、重启了。

稳态量着完美的电压，为什么负载一跳就砸出一个坑？这个坑，就是这篇的主角——**电源瞬态响应（Load Transient Response）**。

**读完这篇，你能搞懂：**

- ✅ 什么是**负载阶跃**，为什么芯片"待机→满载"那一瞬最危险；
- ✅ 电压为什么会先**砸一个坑（undershoot）**、卸载又**冲一个峰（overshoot）**；
- ✅ 这一下到底是谁在硬扛——**稳压器的反应速度**和**输出电容**的真实分工；
- ✅ 它和我们讲过的**压降 / PDN（07）**、**IR drop（11）**到底有什么不一样；
- ✅ 怎么用**电子负载 + 示波器**把这个"坑"真正测出来、又怎么判断合不合格；
- ✅ 坑太深了怎么救：加电容、改补偿、还是换稳压器。

## 一、谜题：稳态合格，是因为你没测"那一下"

先把答案直接说了：**你的万用表和稳态波形，测的都是"电流不变"时的电压；而芯片重启，发生在"电流突变"的那一瞬间——那一下，你根本没测。**

芯片不是匀速吃电的。CPU 待机时可能只吃 **0.5A**，可游戏一加载、睿频一拉满，电流可能在**几微秒内猛跳到 10A**。这种"电流的急刹车/急加速"，工程师叫它**负载阶跃（load step）**。

问题来了：**稳压器（给芯片供电的那颗 DCDC 或 LDO），反应是有延迟的。** 它靠一个**反馈环路**工作——发现电压掉了，才调整、把电压补回来。但这个环路的"反应速度"有限（带宽通常就几百 kHz），**从发现掉压到补回来，要花几微秒**。

而芯片猛抽电、就发生在这几微秒里。**稳压器还没反应过来，电流已经涨上去了**——这中间的"空窗期"，电压没人管，自然就被拽下去，砸出一个坑。

> **一句话记住：万用表测的是"电流稳定时"的电压；而芯片重启，发生在"电流突变"的那几微秒里——稳压器还没反应过来，电压就先掉坑了。稳态全合格，恰恰是因为你没测"突变那一下"。**

## 二、那几微秒，谁在硬扛？输出电容的"救场"

既然稳压器要几微秒才反应过来，那**在它反应过来之前，电压是怎么没有直接掉到 0、芯片没有立刻断电的**？

靠的是稳压器输出端那些**输出电容（output/bulk capacitor）**。它们就像供电路上的一个个**小水库**：负载突然猛抽电、稳压器还没供上的那一瞬，**先由输出电容把存的电"放"出来顶上**，硬扛过这段空窗期。

但电容放电，自己的电压就会往下掉——这就是那个**坑（undershoot / droop）**的来历：

1. **负载阶跃**：电流瞬间从 0.5A 跳到 10A；
2. **空窗期**：稳压器还没反应，全靠输出电容放电顶着 → 电容电压往下掉 → **电压砸出一个坑**；
3. **环路追上**：几微秒后稳压器反应过来、加大输出 → 把电压**拉回** 1.0V；
4. **卸载反向**：游戏加载完、电流又猛地掉回 0.5A，稳压器一时"刹不住车"、还在大量供电 → 多出来的电把电压**顶起一个峰（overshoot）**，再慢慢回稳。

所以一次完整的负载突变，电压波形是：**先砸一个坑，再回弹、甚至冲一个峰，然后慢慢稳定**。坑有多深、峰有多高、多久回稳——这几个数，就是"瞬态响应"好不好的全部。

> **一句话记住：负载突变那几微秒，是输出电容这个"小水库"先放电硬扛、把电压坑撑住的；等稳压器反应过来才把电压拉回。所以瞬态响应，本质是"稳压器反应速度"和"输出电容够不够"的一场接力。**

## 三、它和「压降 / PDN」「IR drop」到底不一样在哪？

讲到这，老读者可能会问：这听起来，和第 07 篇的**压降 / PDN**、第 11 篇的 **IR drop**，不是一回事吗？都是"电压掉了"啊。

**不是。这是三个不同的视角，别混。** 一句话各自钉死：

- **IR drop（11）**：**稳态、直流**。电流稳定地流过有电阻的导线/平面，压一个固定的降。它是"**路一直堵着，水流过去就矮一截**"，和时间、突变都无关。
- **压降 / PDN（07、09）**：**频域**。把供电路看成一条随频率变化的**阻抗曲线 Z(f)**，重点是**高频段**靠去耦电容压低阻抗——管的是芯片内部那种**纳秒级的超快电流波动**。
- **瞬态响应（13，本篇）**：**时域、一次性的负载阶跃**。重点是**稳压器环路的"反应速度"**，以及**输出电容**能不能扛过它反应过来之前的那**几微秒空窗期**。

打个比方：你拧开水龙头猛接一桶水——
- **IR drop** = 水管细，水压一直比理论值低一点（恒定）；
- **PDN / 去耦** = 应对"开关阀门时那种极快的水锤"，靠管路上的小气囊（去耦电容）缓冲；
- **瞬态响应** = 你**突然把出水量开到最大**，水箱浮球阀（稳压器）来不及补水，水位先"咚"地掉下去一截，等阀门反应过来才补上——**那一下的掉水位**，就是瞬态响应。

> **一句话记住：IR drop 是"稳态直流压降"，PDN/去耦是"频域看高频阻抗"，瞬态响应是"时域看稳压器对一次负载突变的反应速度"。三者是同一套供电系统的三个不同侧面，测法和判据都不同。**

## 四、怎么把这个"坑"测出来？

要测瞬态响应，关键是**得能制造出"负载突变"，再把那一瞬间的电压坑抓下来**。靠两件设备配合：

**1. 电子负载（动态模式）—— 制造那一下负载阶跃。**
普通测试用恒定电流负载，但瞬态测试要用电子负载的**动态 / 瞬态模式（dynamic / transient mode）**，给它设三个参数：
- **阶跃幅度 ΔI**：从多少跳到多少（如 0.5A → 10A）；
- **压摆率 di/dt**：跳得多快（如 2.5 A/µs）——**这个越快，坑越深，是最考验稳压器的设定**；
- **频率 / 占空比**：反复加载/卸载的节奏，方便示波器稳定触发。

**2. 示波器（AC 耦合 + 放大）—— 把毫伏级的坑看清楚。**
电压坑通常只有几十毫伏，趴在 1.0V 上直接看根本看不清。技巧：
- 用 **AC 耦合**，把 1.0V 的直流"台面"去掉，只留波动，再把纵轴放大到几十 mV/格——坑就一下子撑满屏幕；
- **探头要探到最靠近芯片的地方**（回扣第 07 篇：电容脚合格 ≠ 芯片嘴合格，瞬态更要探到芯片真正吃电处）；
- 探头要**够带宽、地线要短**（第 03 篇讲过：地线一长，振铃和过冲全是假象）；
- 触发设在负载阶跃的边沿，把 undershoot 和 overshoot 都抓全。

> **一句话记住：瞬态响应＝用电子负载动态模式造一次"急加载"（设 ΔI、di/dt、频率），再用示波器 AC 耦合放大、探到芯片最近处，把那个几十毫伏的电压坑和峰完整抓下来。**

## 五、测出来了，怎么判断合不合格？

抓到波形后，对着三条线看：

1. **坑深 / 峰高 ΔV：必须在芯片的容差窗口内。** 芯片 datasheet 会给供电的容差，比如 1.0V ±5%，那下限就是 **0.95V**。只要 undershoot 的最低点**没跌破 0.95V**、overshoot 的最高点没超过上限，就算扛住了。我们开头那块板子掉到 0.90V，**跌破了下限，所以重启**——这就是不合格。
2. **恢复时间（settling time）：回稳要够快。** 从掉坑到稳定回到容差带内，时间不能太长，否则芯片可能在这段"半饱"状态里出错。
3. **不能振铃（ringing）：回稳过程不能来回振荡。** 如果电压拉回时上下来回荡好几下才停，说明稳压器的**环路稳定性**有问题（相位裕度不够）——这是另一篇"环路稳定性"的重头戏，这里先记住：**好的瞬态响应是"掉坑—利落拉回"，不是"掉坑—荡半天"。**

> **一句话记住：瞬态响应合格 = 坑深和峰高都在容差窗口内 + 回稳够快 + 不来回振铃。三条任一不满足，芯片都可能在负载突变那一下出错。**

![图7：瞬态判据三条线——ΔV 在容差窗口内、恢复时间够快、不振铃（合格 vs 不合格对照）](/images/13/ill07_criteria.png)

## 六、坑太深了怎么救？兼说 DCDC 与 LDO 的区别

如果测出来坑跌破了下限，怎么救？三个方向（细节是后续专门篇的内容，这里给方向）：

1. **加 / 优化输出电容**——多放、放对位置、选低 ESR 的电容，让"小水库"更能扛那几微秒，**直接填浅坑**。这是最常用的一招。
2. **改环路补偿、提高带宽**——让稳压器"反应更快"，空窗期更短。但带宽不能无限提，提过头会**振铃、不稳定**（又回到环路稳定性）。
3. **换一颗瞬态响应更好的稳压器**——从源头选反应快、规格好的电源 IC。

这里正好点出 **DCDC 和 LDO 在瞬态上的区别**（详见后续"电源 IC 家族总览"）：
- **DCDC（开关电源）**：效率高、能带大电流，但**反应相对慢**（受开关频率/环路限制），大电流瞬态主要**靠足够的输出电容**扛；
- **LDO（线性稳压器）**：**反应快、输出干净**（噪声小，适合 RF/时钟这类敏感供电），但**带不了大电流、效率低**。

所以选谁，得看这一路供电是"**大电流、要效率**"（用 DCDC，配好输出电容）还是"**小电流、要干净/反应快**"（用 LDO）。

> **一句话记住：救瞬态坑三招——加输出电容(填坑)、改环路补偿提带宽(让稳压器更快，但小心振铃)、换更好的稳压器。DCDC 带大电流但反应慢靠电容扛，LDO 反应快但带不了大电流。**

## 七、一页速查：电源瞬态响应要点

| 关键词 | 一句话本质 | 新手常踩的坑 |
|--------|-----------|-------------|
| 负载阶跃 | 电流瞬间突变(待机→满载) | 只测稳态、不测突变那一下 |
| undershoot 坑 | 稳压器没反应过来，电压被拽下去 | 以为稳态合格就万事大吉 |
| overshoot 峰 | 卸载时稳压器刹不住，电压顶上去 | 只看掉坑、忘了卸载冲高 |
| 输出电容 | 那几微秒靠它放电硬扛 | 电容不足/选错，坑填不住 |
| 环路反应速度 | 带宽决定多快补回来 | 以为稳压器能瞬间响应 |
| di/dt | 阶跃越快，坑越深 | 测试时 di/dt 设太慢，测不出真坑 |
| AC 耦合放大 | 去掉直流台面看毫伏级坑 | 直流耦合直接看，啥也看不清 |
| 探近芯片 | 探到芯片真正吃电处 | 探电容脚当芯片电压 |
| 判据三条 | ΔV 在窗口+回稳快+不振铃 | 只看一条，漏判 |

## 写在最后

我们从那个抓狂的谜题出发——"稳态全合格，一加载就重启"——现在可以收网了：

**因为你的万用表和稳态波形，测的都是"电流不变"时的电压；而芯片重启，发生在"电流突变"的那几微秒里。负载猛地跳起来、稳压器还没反应过来的那个空窗期，全靠输出电容放电硬扛，电压先被拽出一个坑——这个坑要是跌破了芯片的容差下限，它就在那一下饿晕、重启。** 稳态测得再漂亮，都看不见这一下。

> **电源瞬态响应第一准则：别只测"电流稳定时"的电压，要测"电流突变那一下"。把稳压器的反应速度和输出电容这场接力测出来——坑深、峰高、回稳快慢、有没有振铃，才是供电稳不稳的真考题。**

### 判据表 1 · 瞬态响应关键指标与判据

| 指标 | 含义 | 典型判据 |
|------|------|---------|
| ΔI 阶跃幅度 | 负载电流突变量(如 0.5A→10A) | 按芯片真实满载电流设 |
| di/dt 压摆率 | 阶跃速度 | 常用 1–2.5 A/µs 起，按 datasheet 上限测 |
| Undershoot(坑深) | 加载瞬间最低电压 | 不得跌破供电下限(如 1.0V±5% → ≥0.95V) |
| Overshoot(峰高) | 卸载瞬间最高电压 | 不得超过供电上限(≤1.05V) |
| Recovery/Settling time | 回到容差带内的时间 | 越短越好，按芯片要求(常 µs 级) |
| 振铃 ringing | 回稳是否振荡 | 不应来回振荡(→查环路相位裕度) |

### 测试设置（示波器 / 电子负载）
- **电子负载**：动态/瞬态模式，设 ΔI、di/dt(slew rate)、频率/占空比；阶跃用方波。
- **示波器**：**AC 耦合**、纵轴放大到 10–50 mV/格；带宽≥被测信号 5×；触发设在负载边沿。
- **探头**：探到**最靠近芯片的输出电容/芯片供电脚**；**地线尽量短**(弹簧地/同轴)，避免假振铃。
- 同时抓**负载电流(电流探头)**与**输出电压**两路，对齐看因果。

### 器件差异（瞬态视角）
- **DCDC(开关)**：大电流、高效率；环路受开关频率限制，反应较慢，大瞬态**主要靠输出电容**扛；注意 SW node ringing。
- **LDO(线性)**：反应快、输出噪声低(高 PSRR)，适合 RF/时钟/ADC 等敏感供电；但**带不了大电流、效率低、压差损耗大**。
- **选型**：大电流要效率→DCDC+足量输出电容；小电流要干净/快→LDO。

### 改善瞬态响应的手段
- 增加 / 优化**输出电容**(容值、低 ESR、靠近负载)——最直接，填坑。
- 提高环路**带宽**/优化**补偿**——缩短空窗期，但过头会振铃(权衡相位裕度)。
- 选**瞬态规格更好**的稳压器；必要时加**本地 LDO/Load Switch** 分级供电。

### 踩坑 checklist
- [ ] 别只测稳态；必须用电子负载动态模式测"突变那一下"。
- [ ] di/dt 要按 datasheet 上限设，太慢测不出真坑。
- [ ] 示波器用 AC 耦合放大，否则几十 mV 的坑看不见。
- [ ] 探近芯片、地线短，否则振铃/过冲是假象。
- [ ] 判据三条都要看：ΔV 窗口 + 回稳时间 + 不振铃。
