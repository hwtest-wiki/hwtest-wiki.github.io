# IP68 防水，为什么手机掉水里还是进水了？

你的手机，大概率标着「IP68 防水」。可你真的清楚它防的是哪种水、能泡多久、万一进水保不保修吗？大多数人，其实从没真正读懂过这串代码。

先讲个真事。

2022 年 6 月，三星澳洲被罚了 **1400 万澳元**（约合人民币 6500 万元）。原因不是手机炸了，而是**广告**：三星多年宣传旗下 Galaxy 手机「可以在泳池、海水里用」，配着冲浪、泡水的画面。可实际上，这些手机**在海水或泳池水里浸泡后会受损，湿着插线充电还会腐蚀充电口**。最后澳大利亚联邦法院认定它**误导消费者**，开出这张罚单（来源：澳大利亚竞争与消费者委员会 ACCC，2022-06-23）。

问题来了：那些手机，**明明都标着 IP68 防水**啊。

这就是防水最大的认知陷阱：**两台都写着「IP68」的设备，一台你能放心带它下雨天用，另一台一进水就报废——它们的标签一模一样，结局却天差地别。**

这一篇，我们就把三件事彻底讲透：**①「IP68」这串代码到底什么意思；② 手机内部到底用了什么手段把水挡在外面；③ 你怎么判断自己手里这台手机，到底防不防水。**

**读完这篇，你能搞懂：**

- ✅ **IP68 这串代码怎么读**，防尘、防水各看哪一位；
- ✅ 最容易混的 **IPX7 / IPX8 / IPX9K**，差在水深还是水压；
- ✅ **手机内部到底怎么防水的**：胶圈、点胶、防水透气膜、纳米涂层；
- ✅ 为什么**标着防水的手机，照样会进水**（三星那 1400 万就栽在这）；
- ✅ 实验室里**到底怎么测防水**，浸水、喷淋、高压、检漏各是什么；
- ✅ **怎么判断你自己的手机防不防水**，以及进水了第一时间该怎么办；
- ✅ 大家最爱问的几个问题：**大米能不能救手机、5ATM 和 IP68 哪个能游泳、防水袋有没有用**。

## 一、先把「IP68」这串代码拆开

我们天天看到「IP67」「IP68」「IPX4」，但很少有人真正读懂它。其实它的结构非常规整：

> **IP + 第一位数字 + 第二位数字**

- **IP**：是 **Ingress Protection（防护进入）** 的缩写，它对应一份国际标准——**IEC 60529**（国内等同 GB/T 4208）。
- **第一位数字（0–6）**：防**固体**，主要就是**防尘**——挡灰尘、也挡你手指碰到内部带电部件。
- **第二位数字（0–9 / 9K）**：防**液体**，也就是我们最关心的**防水**。

所以「IP68」= **防尘等级 6（最高）+ 防水等级 8**。而你常见的「IPX4」里那个 **X**，意思是「防尘这项没测、不声明」，**不是 0**。

防尘这一位，从 0 到 6 越来越严：能挡 50mm 的手背、挡 1mm 的铁丝，到 **IP5X**（进尘量不影响工作）、**IP6X**（一粒灰都不进，叫"尘密"）。手机普遍做到最高的 **6**，所以 IP67、IP68 第一位都是 6，**大家真正较劲的，是后面那个防水的数字**。

> **一句话记住：IP 后面两位，第一位管防尘（0–6）、第二位管防水（0–9K），各自独立分级、数字越大越强。看手机防水，只盯第二位。**

## 二、第二位：防水，从「淋几滴」到「高压热水冲」

重头戏来了。第二位数字（防水）的跨度非常大，**关键在于：它不是简单的「数字越大泡越久」，而是测试的"场景"完全不同。** 我们按从弱到强捋一遍：

| 等级 | 防什么 | 大白话 |
|------|--------|--------|
| IPX1 / IPX2 | 垂直 / 小角度滴水 | 天花板渗水、毛毛雨 |
| IPX3 | 60° 范围内淋水 | 正常下雨、花洒斜着冲 |
| IPX4 | 各方向**泼溅** | 泼水、雨里走、出汗，不怕 |
| IPX5 | 各方向**低压喷射**（6.3mm 喷嘴） | 水龙头冲、被水枪扫到 |
| IPX6 | 各方向**强力喷射**（12.5mm 喷嘴） | 甲板、消防水柱级别冲洗 |
| IPX7 | **浸入 1 米水中 30 分钟** | 掉进水盆、马桶捞得起来 |
| IPX8 | **浸入比 1 米更深**（厂商定义，常见 1.5m / 30min 起） | 厂家自己标的更深浸泡 |
| IPX9 / IPX9K | **高温高压水柱冲洗**（80℃、80–100bar） | 工业高压清洗机直喷 |

这里有两个**反直觉**、却最容易踩坑的点，必须挑明：

**第一，「喷」（5/6）和「泡」（7/8）是两条线，不是一条。** IPX6 能扛高压水柱，但它测的是"冲"、不是"泡"；理论上一个只过了 IPX6 的设备，**不保证泡进水里不漏**。反过来 **IPX8 也不自动等于过了 IPX6**。所以严谨的工业设备会写成 **IP66/IP68 两个等级**，把「能冲」和「能泡」分开证明。

**第二，手机厂宣传的「IP68」，深度其实是厂商自己定的。** 因为标准里 IPX8 没有写死统一数字，只要求「比 IPX7 更严、深度时长由厂商约定」。这就是为什么同样 IP68，有的标 1.5 米、有的标 2 米、时间也不一样——**同样三个字符，能泡的本事并不相同。**

而手机圈最常混的三个，单独说清楚：

- **IPX7：浸入 1 米清水、30 分钟。** 固定、明确的标准动作——清水、常温、静止。
- **IPX8：比 IPX7 更深，但"多深、多久"由厂商定义。** 你看到的「支持 1.5 米 30 分钟」就是厂商自己测、自己标的。
- **IPX9 / IPX9K：80℃ 高温、80–100bar 高压水柱近距离冲。** 给汽车、厨房、户外重型设备用的，**核心是"高压冲"不是"泡得更深"**，和 7/8 根本不是一回事。

> **一句话记住：防水第二位不是「时间越长级别越高」，而是测试场景不同——5/6 是「冲」、7/8 是「泡」、9K 是「高温高压冲」。喷得住不代表泡得住，而手机的 IP68 深度是厂商自己定的。**

## 三、手机到底是怎么做到防水的？

手机不像潜水表那样厚墩墩、还能拧螺丝压胶圈，它薄得只有几毫米，却要塞进屏幕、电池、摄像头，机身上还开了一堆孔：充电口、SIM 卡槽、听筒、喇叭、麦克风。**防水的难点，恰恰就是这些"非堵不可、却又必须留着"的缝和孔。** 工程师用的是一套"组合拳"，分三道防线：

**第一道·把缝封死（密封圈 + 点胶）。**
屏幕、中框、后盖这几大块拼在一起，接缝处要么压一圈**橡胶密封圈**，要么打一圈**防水胶/背胶（点胶工艺）**，把缝隙彻底堵住。SIM 卡托上那圈橡胶圈，也是同一个道理——你拔卡时能摸到的那圈软胶，就是防水件。

**第二道·让该开的孔"只透气声、不透水"（防水透气膜）。**
听筒、喇叭、麦克风这些孔，**必须让声音和空气通过，又不能让水进去**，怎么办？贴一层**防水透气膜**（常见材料是 ePTFE，类似冲锋衣 Gore-Tex 那种微孔薄膜）。它的微孔**比空气分子大、比液态水的水珠小**：声音、空气能穿过去，而水因为**表面张力**会在膜外面"绷"成水珠、钻不进那些微孔。同样的膜还会做成一个**气压平衡阀**——手机完全密封后，温度一变（比如从空调房到户外、洗澡蒸汽里），机身内外会产生**气压差**，这个膜能放气、平衡压力，避免把水"吸"进去或把密封顶坏，依然只透气、不透水。

**第三道·万一进了一点也不立刻死（板级纳米涂层）。**
前两道是"挡在门外"，第三道是"兜底"：在电路板（PCBA）上喷一层**三防漆 / 纳米防水涂层**（比如等离子镀膜工艺），给元件穿层极薄的"雨衣"。这样即使有极少量水汽渗进来，关键电路也不会马上短路。**注意：这是延缓、不是免疫**——大量进水、泡海水照样完蛋。

这三道防线叠起来，才换来手机那张「IP68」的成绩单。也正因为它靠的是**胶圈、薄膜、涂层**这些会老化、会被外力破坏的东西，所以防水能力**不是终身不变的**——这一点，后面还会反复提到，也是很多人吃亏的地方。

> **一句话记住：手机防水＝三道防线——胶圈/点胶把缝封死、防水透气膜让孔"只过气声不过水"、板级纳米涂层兜底。全是会老化、怕磕碰的东西，所以防水会随使用衰减。**

## 四、为什么"标着防水"，照样进水？三星那 1400 万的真相

现在回到开头那个 1400 万的罚单。手机标着 IP68，为什么还是被泡坏、还被法院判误导？因为 **IP 测试是一套"实验室特定条件"，和你日常用水的真实世界，差着十万八千里**。具体差在哪：

1. **测的是清水，不是海水 / 泳池水。** IEC 60529 用的是**洁净的常温自来水**；而**海水含盐、泳池水含氯**，腐蚀性强得多——三星栽的就是这条：广告让你下海、下泳池，可那根本不在测试条件里。
2. **测的是静止的水，不是流动 / 冲击的水。** 浸泡测试里水是静止的；而你游泳、冲浪时，水**高速流动、带冲击压力**，瞬时压强远超「1 米静水」，更容易从缝隙灌进去。
3. **测的是常温，不是忽冷忽热。** 桑拿、热水、温差会让机身内外产生**气压差**，把水"吸"进去——这就是说明书常写「不要在洗澡、桑拿时用」的原因。
4. **测的是出厂的新机，不是用了一年的旧机。** IP 等级是**出厂那一刻、在实验室**测的。手机摔过、密封胶圈老化、后盖被拆修过之后，防水能力会**断崖式下降**。
5. **湿着充电 = 自己开门放水。** 三星案例特别点名这条：**充电口湿着就插线充电，会腐蚀触点**。防水是"挡水进去"，可你主动把湿的金属通上电，那是另一回事。

所以「IP68」真正的含义，是一句**带很多前提的承诺**：「在实验室、用清水、常温、静止、出厂状态下，泡到厂商声明的深度和时间不进水」。它**绝不等于**「随便什么水、随便泡多久、用多久都没事，坏了还包修」。

> **一句话记住：IP 等级是"实验室清水、常温、静止、出厂新机"的成绩单，不是"任何水、任何时候随便泡"的免死金牌。海水、泳池、热水、老化、带水充电，全在保单之外。**

![图7：实验室条件（清水·常温·静止·新机）vs 真实世界（海水·氯·高温·冲击·老化·带水充电）](/images/12/ill07_lab_vs_real.png)

## 五、实验室里到底怎么测防水？四类方法一次看懂

那这张「IP68」成绩单，工程师在实验室里**具体怎么测出来**？按场景，主要是四类：

1. **浸水测试（测 IPX7 / IPX8）——浸水箱 / 压力水槽。** 把设备按规定深度沉入清水，计时 30 分钟（或厂商时长），到点取出、擦干、通电检查有没有进水、功能是否正常。IPX8 还会用**可调压力水箱**模拟更深的水压。
2. **喷淋 / 喷射测试（测 IPX3–IPX6）——摆管 / 喷头。** IPX3/4 用**摆管**（半圆形带很多小孔、左右摆动淋水）；IPX5/6 用**标准喷嘴**（6.3 / 12.5mm 内径），按规定水压、水量、距离各方向冲。
3. **高温高压测试（测 IPX9K）——专用高压热水喷枪。** 80℃ 热水、80–100bar，从 0°/30°/60°/90° 四个角度近距离冲，主要用于车规、工业设备。
4. **气密性 / 检漏测试（量产线上的"防水体检"）——压差检漏仪。** 浸水测试要泡水、效率低，**不适合每台都做**。所以产线上更多用**气密性测试**：给设备内腔充一点气，测**气压在几秒内掉了多少**——掉得快说明有漏点。它**间接、快速**地反映密封好坏，是量产防水的主力测法。

> **一句话记住：浸水箱测"泡"（7/8）、摆管喷头测"淋和喷"（3–6）、高压热水枪测 9K；而产线上每台都做的，是用"充气测压降"的气密检漏来快速筛漏点。**

## 六、怎么知道你自己的手机防不防水？

讲了这么多，落到最实在的问题：**你手里这台手机，到底防不防水、能不能放心带它淋雨？** 给你一套可操作的判断法：

1. **先查 IP 等级——看官网规格页，别信感觉。** 去手机**官网的"技术规格"页**，或包装盒/说明书里，搜「机型 + IP」。**标了 IP67 / IP68** 才有防水设计；标 **IPX4** 只防溅防汗、不能泡；**完全没标**，那就当它**不防水**。
2. **分清"能泡"还是"只防溅"。** IP67 / IP68 才敢短时浸水（按厂商深度）；IPX4 那类，下点小雨、出汗没事，**别往水里放**。
3. **记住最扎心的一条：防水 ≠ 保修。** 国内外**绝大多数厂商，进水都不在保修范围内**。手机内部（常在 SIM 卡槽里）有一片白色的**液体接触指示器（LDI）**，沾水会变红——售后一看变红，基本就判定"进过水"、拒绝保修。**IP 是"设计能力"，不是"质保承诺"。**
4. **旧机、摔过、拆修过的，请直接给防水打折。** 防水靠的是会老化的胶圈和涂层：手机用了一两年、摔过、进过水、换过屏 / 换过电池（除非维修时重新做了防水点胶和密封），它的防水能力**很可能已经名存实亡**，别再拿当年的 IP68 当护身符。
5. **千万别"自己泡水验证"。** 拿自己的手机泡水测试，等于拿手机和保修去赌——标准测试是清水、静止、特定深度时间，你在家很难复现，**翻车概率极高，还没法索赔**。想知道防不防水，查规格，不要实测。
6. **万一进水了，第一时间这么做：** **立刻关机**、**别充电**、**别反复开机**，擦干外部、取出 SIM 卡托晾干；**别用吹风机热风吹**（高温会伤密封、还可能把水吹进更深）；可以放通风处或干燥剂旁自然晾，**尽快送修**。越早断电，烧坏主板的概率越低。

> **一句话记住：查官网 IP 等级定性质（IP67/68 能泡、IPX4 只防溅、没标当不防水）；牢记进水多半不保修（看 LDI 变没变红）；旧机摔过拆修过一律打折；别自己泡水测；进水先关机别充电、尽快送修。**

## 七、大家最爱问的几个防水问题（一次说清）

这一节，把网上关于手机防水问得最多、也错得最多的几个问题，一口气讲明白：

**🍚 Q1：手机进水，埋大米里能救吗？**
**基本没用，还可能帮倒忙。** 大米吸的是空气里的潮气，吸得又慢又弱，根本"抽"不出手机内部的水；更糟的是**米粒、淀粉碎屑会钻进接口**。苹果官方都明确**反对**用大米。正确做法：**立刻关机、拔掉线、开口朝下轻拍甩水，放在通风处自然晾**；想加速就用**硅胶干燥剂**（比大米强得多）；**至少等 5 小时、彻底干了再开机充电**；**别用吹风机热风**（高温伤密封、还把水吹更深）；不放心就尽快送修。

**🔊 Q2：喇叭 / 充电口进水变闷，怎么把水弄出来？**
靠**声波 + 振动**把水"震"出来。Apple Watch 的「水锁」功能就是这原理——转动表冠播放一段特定声音，让扬声器振动把孔里的水顶出去；安卓也有类似的"排水"小程序。配合**开口朝下轻轻甩**效果更好。**千万别拿棉签、别针去捅**，容易把防水网膜捅破。

**⌚ Q3：手表写「5ATM」、手机写「IP68」，哪个更能游泳？**
这是**两套完全不同的体系**，不能直接比大小。**ATM 是"静水压"**——5ATM 约等于能扛 50 米深的**静止水压**（符合手表标准 ISO 22810 才适合游泳）；**IP 是 IEC 60529 那套场景测试**。**而且"数字大"不等于"能潜水"**：5ATM、IP68 都只适合日常和游泳，**跳水、潜水、高速水上运动都不行**；真要潜水，得认准标着「Diver's」、符合 ISO 6425 的专业潜水表。

**🛍️ Q4：防水袋、防水壳、防水喷雾，哪个真有用？**
**合规的 IPX8 防水袋、潜水壳是真有用的**——套好了甚至能水下拍照。但**防水喷雾、所谓"纳米防水贴膜"对"泡水"基本没用**，最多防点小溅。要带手机下水，老老实实用防水袋 / 潜水壳。

**⚡ Q5：湿着手、手机没干，能充电吗？**
**别。** 接口带水充电会**腐蚀触点**，还有安全风险。现在很多手机检测到充电口有液体，会**自动暂停充电并弹窗提示**——这时硬充就是和手机过不去。等**彻底干透（几小时）**再插线；急用可以先用**无线充电**应急。

**🚿 Q6：防水手机能戴着洗澡、泡温泉吗？**
**强烈不建议。** 一是**热水的温差**会让机身内外产生气压差，把水往里"吸"；二是更隐蔽的一点——**肥皂、洗发水、沐浴露会大幅降低水的表面张力**，而手机喇叭那层「防水透气膜」**正是靠水的表面张力挡水的**，表面张力一垮，水就容易钻进去。所以"清水溅一下没事"和"带泡泡水里泡澡"，完全是两回事。

> **一句话记住：大米救不了手机（关机+干燥剂+耐心），喇叭进水靠声波震出来；5ATM 和 IP68 是两套体系、都不能潜水；防水袋有用、防水喷雾没用；湿着别充电；带肥皂的热水最伤防水。**

## 八、一页速查：防水等级与判断要点

| 关键词 | 一句话本质 | 新手常踩的坑 |
|--------|-----------|-------------|
| IP + 两位数 | 第一位防尘(0–6)、第二位防水(0–9K) | 把两位当一个整体数字看 |
| X 的含义 | 该项没测/不声明，不是 0 | 把 IPX4 当成 IP04 |
| 喷 vs 泡 | 5/6 是冲、7/8 是泡，两条线 | 以为过了 IPX6 就一定能泡 |
| IPX8 | 深度时间由厂商定义，不统一 | 以为所有 IP68 泡水能力一样 |
| 手机怎么防水 | 胶圈点胶 + 防水透气膜 + 纳米涂层 | 以为是「整块密封铁板」 |
| 防水会衰减 | 胶圈涂层会老化、怕磕碰 | 旧机/摔过还当新机泡 |
| 清水 ≠ 海水 | 测试用常温静止清水 | 拿去泡海水/泳池/热水 |
| 防水 ≠ 保修 | 进水多不保修，看 LDI | 以为防水坏了能包修 |
| 进水急救 | 关机、别充电、尽快送修 | 湿着充电 / 热风吹 |

## 写在最后

我们从那张 1400 万的罚单出发，现在可以收网了：

**「IP68」不是一句「能防水」的口号，而是一份带满前提的实验室成绩单——清水、常温、静止、出厂新机，泡到厂商声明的深度和时间不进水。** 手机为它在内部布了三道防线：胶圈点胶封缝、防水透气膜守孔、纳米涂层兜底；可这些东西都会老化、怕磕碰，所以防水会随使用慢慢衰减。看懂第二位数字测的是哪种水、记住"防水不等于保修"、给旧机打个折，你就不会再被"防水"两个字忽悠，也不会真把手机带去海里冲浪。

> **防水测试第一准则：IP 等级是"特定实验室条件、出厂新机"下的承诺，不是"任何水、任何时候、用多久"的保证。读懂第二位测的是哪种水、知道手机靠什么防水、清楚它会衰减——比记住「它防水」重要一百倍。**

### 手机防水的三道防线（结构 / 工艺）
- **密封**：屏/中框/后盖接缝处**密封胶圈 + 点胶（背胶）**；SIM 卡托橡胶圈。
- **开孔**：听筒/喇叭/麦克风贴**防水透气膜（ePTFE）**——微孔透气透声、靠表面张力挡水。
- **气压平衡**：**透气阀/膜**平衡温差气压差，只透气不透水。
- **板级兜底**：PCBA 喷**三防漆 / 纳米防水涂层（如等离子镀膜）**，延缓少量进水短路（非免疫）。
- **接口**：USB-C 内部密封、防水连接器。

### 常见设备 IP 等级参考
- 旗舰手机：多为 **IP68**（厂商常标 1.5 m / 30 min，个别更深）。
- 普通防水手机 / 耳机：**IP67**（1 m / 30 min）或 **IPX4 / IPX5**（仅防溅 / 防汗）。
- 户外 / 三防设备：常见 **IP67 / IP68 + IP6K9K** 双标。
- 工业 / 车规连接器：**IP67 / IP69K** 常见。

### 测试方法与设备
- **浸水箱 / 可调压水槽**：测 IPX7 / IPX8。
- **摆管（oscillating tube）+ 标准喷嘴（6.3 / 12.5 mm）**：测 IPX3–IPX6。
- **高压热水喷枪（80℃ / 80–100 bar）**：测 IPX9 / IPX9K。
- **气密性 / 压差检漏仪**：产线全检主力，测充气后压降速率，间接判密封。
- 浸水后判定：取出擦干 → 通电 → 查进水痕迹与功能，**不得有影响功能的进水**。

### 用户自查 checklist（怎么知道自己手机防不防水）
- [ ] 查**官网技术规格**页确认 IP 等级；没标当不防水。
- [ ] 分清能泡（IP67/68）还是只防溅（IPX4）。
- [ ] 记住**进水多半不保修**；看 SIM 槽内 **LDI（液体接触指示器）**是否变红。
- [ ] **旧机 / 摔过 / 进过水 / 换过屏电池**的，防水能力按"已失效"对待。
- [ ] **不要自己泡水实测**（清水静止特定条件难复现，翻车不赔）。
- [ ] 进水急救：**关机、别充电、别反复开机、别热风吹、尽快送修**。

### 常见疑问速答（大众高频）
- **大米救手机**：基本无效且可能进米粒/淀粉；苹果官方反对。正确：关机→开口朝下甩→通风晾→硅胶干燥剂→≥5h 再开机→别热风→送修。
- **喇叭/口进水**：声波+振动排水（Apple Watch「水锁」原理；安卓有排水 App）；别用尖物捅破透气膜。
- **5ATM vs IP68**：两套体系。ATM=静水压（5ATM≈50m 静压，ISO 22810 适合游泳）；IP=IEC 60529 场景测试；二者均不适合跳水/潜水（潜水认 ISO 6425「Diver's」）。
- **防水袋/壳**：合规 IPX8 防水袋、潜水壳有效（可水下拍照）；防水喷雾/贴膜对泡水基本无效。
- **湿手充电**：禁，腐蚀+风险；多数手机检测到接口液体会暂停充电；干透或先无线充。
- **洗澡/温泉**：禁。热水温差吸水；皂液/洗发水降低水表面张力→喇叭防水透气膜失效。

- **ISO 20653**（道路车辆，定义 **IPX9K** 高温高压喷水）。
