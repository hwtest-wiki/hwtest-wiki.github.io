# 电源没坏却"吱吱"啸叫、输出还规律抖动——稳压器怎么自己变成了振荡器？

有没有遇到过这种怪事：电源每个元件都好好的，却莫名其妙发出"吱吱"的啸叫，输出电压还规律地上下抖动？这往往不是哪里坏了，而是稳压器把自己"逼"成了一个振荡器。

先讲个真事。

上一篇那块板子，工程师嫌电压坑太深、瞬态太慢，于是动手"提速"：**减小输出电容、把环路带宽往上拉**，想让稳压器反应更快。

坑确实变浅了。可新问题来了：电源开始发出**轻微的"吱——"的啸叫**，示波器一看，输出电压不再是平的，而是**规律地上下抖动，一条幅度稳定的正弦波**——这不是纹波，是**等幅振荡**。

明明每个元件都是好的，**稳压器怎么自己变成了一个"振荡器"？**

答案，藏在电源完整性里最让人头大、却又最优雅的一个概念里——**环路稳定性（Loop Stability）**，以及衡量它的那把尺子：**相位裕度（Phase Margin）**。这一篇，我们就用几个生活里的例子，把它彻底讲明白。

**读完这篇，你能搞懂：**

- ✅ 稳压器明明是"负反馈"，为什么会**自己振荡**起来；
- ✅ 为什么"反应太快"反而会出事——**快与稳的矛盾**到底在哪；
- ✅ **相位裕度**是什么，为什么 **-180°** 是一条"生死线"；
- ✅ 工程师怎么用一张**波特图**，一眼看出"这电源稳不稳、离自激还有多远"；
- ✅ 怎么**测**环路稳定性，又怎么从时域的**振铃**反推它；
- ✅ 电源振荡了怎么**救**：补偿、调电容、降带宽。

## 一、谜题：好端端的稳压器，怎么自己抖起来了

先看清楚现象。正常的稳压器输出，应该是一条**平直线**（带一点点小纹波）。可现在它输出的是一条**幅度稳定、来回摆动的正弦波**——电压在标称值上下，规律地、持续地振荡，停不下来。

这种"自己抖起来、还停不下来"的现象，就叫**自激振荡（self-oscillation）**。它往往还伴随**啸叫**：电感或电容在那个振荡频率上跟着机械振动，发出人耳能听到的"吱——"声（很多人听过老电脑、劣质充电器的"吱吱"声，就是这么来的）。

奇怪的是，**每个元件都是好的**。问题不在某个零件，而在它们组成的那个**反馈系统**身上。

> **一句话记住：电源自激振荡，不是某个元件坏了，而是整个反馈环路"失稳"了——它从一个老老实实稳压的系统，变成了一个停不下来的振荡器。**

## 二、为什么"负反馈"会变成"正反馈"？——推秋千与话筒啸叫

这里有个反直觉的地方：稳压器明明是**负反馈**——电压高了就往下压、低了就往上抬，应该越来越稳才对，怎么会越抖越厉害？

关键在两个字：**延迟**。

**先看它本来怎么稳压（推秋千·正常版）。**
想象你在推一个秋千，目标是让它**停在中间**。秋千往左偏了，你就往右推一把；往右偏了，你就往左推。只要你**反应及时**，秋千很快就稳在中间了。这就是负反馈稳压：发现偏差，立刻反向纠正。

**再看它怎么失控（推秋千·延迟版）。**
但如果你的**反应有延迟**——你看到秋千往左、却要等一会儿才推出去。要是这个延迟刚好等于**半个摆动周期**，那麻烦了：你"想往右推"的时候，秋千**已经荡回到右边了**。你这一把"纠正",正好**顺着它的运动方向推**——你以为在"拉回来"，实际在"加把劲"。

于是每一次"纠正"都变成了"助推"，秋千越荡越高。**负反馈，因为延迟，变成了正反馈。** 这就是自激振荡的根。

**还有个人人听过的例子：话筒啸叫。**
把麦克风凑近音箱，立刻"吱——"地尖叫。为什么？音箱的声音被麦克风收进去、放大、再从音箱放出来、又被麦克风收进去……**绕成了一个正反馈环**：只要环路里的放大足够强、相位又对得上，一点点声音就会被无限循环放大，直到尖叫。**电源的自激，和这个话筒啸叫，是一模一样的数学。**

> **一句话记住：稳压器靠负反馈纠偏，但反馈有延迟（相位滞后）。当某个频率上延迟达到"半个周期"——也就是相位滞后 180°，负反馈就翻转成正反馈，纠偏变助推，系统自己振荡起来，就像推秋千踩错了点、话筒凑近音箱啸叫。**

## 三、相位裕度：离"翻车"还有多少安全余量

既然 **相位滞后 180°** 是那条致命的"生死线"，那工程师最关心的就是：**我现在离这条线，还有多远？** 这个"安全余量"，就是**相位裕度（Phase Margin，PM）**。

先补一个关键点：振荡能不能起来，要**同时满足两个条件**——① 相位滞后到了 180°（纠偏变助推）；② 在这个频率上，环路的**放大还够强（增益 ≥ 1）**。只要在"增益还大于 1"的频率范围内，相位没到 180°，振荡就起不来。

所以工程师盯的是这么一个点：**环路增益正好等于 1（也就是 0dB）的那个频率**——过了这个频率，放大不足，再怎么相位对得上也振不起来。在这个"临界频率"上，看**相位离 -180° 还差多少度**，这个差值就是相位裕度。

- **相位裕度 > 45°**：安全，系统稳，瞬态响应干脆利落；
- **相位裕度在 0°~45°**：危险，开始**振铃**——电压回稳时上下荡几下才停（还记得上一篇那个"回稳不能振铃"吗？根就在这）；
- **相位裕度 ≤ 0°**：完了，已经越过生死线，**自激振荡**，就是开头那条停不下来的正弦波。

用**开车转向**再体会一下：方向盘打一把要回正，**阻尼够**（相位裕度足）就一把到位、稳稳的；**阻尼不够**（相位裕度小）就会左右来回修、新手"画龙"（振铃）；**完全没阻尼**，方向就来回甩、彻底失控（自激）。相位裕度，就是这个"阻尼够不够"的量化。

> **一句话记住：相位裕度 = 在"环路增益还等于 1"的那个频率上，相位离致命的 -180° 还剩多少度。大于 45° 安全、不到 45° 振铃、到 0° 就自激。它就是电源"快与稳"之间的安全余量。**

## 四、工程师的"体检表"：一张波特图

要把"增益够不够、相位差多少"一眼看清，工程师用的是一张**波特图（Bode Plot）**——它把环路的**增益**和**相位**，分别画成随频率变化的两条曲线，上下叠着看：

- **上图（增益）**：看那条曲线**从上往下穿过 0dB 线**的频率（叫**增益穿越频率**）——这就是"放大正好等于 1"的临界点；
- **下图（相位）**：在**同一个频率**处，看相位曲线掉到了多少度。它离 -180° 的差，就是**相位裕度**；
- 反过来还有个**增益裕度**：在相位正好等于 -180° 的频率上，看增益比 0dB 低多少（一般要求低于 -6dB 以上才安全）。

所以看一张波特图判稳定，就三步：**找增益穿 0dB 的频率 → 看那里相位离 -180° 多远（相位裕度）→ 余量够不够。** 一张图，电源"稳不稳、离自激还有多远"，全写在上面。

> **一句话记住：波特图把环路的增益和相位画成两条随频率的曲线。看增益穿过 0dB 那个频率、对应的相位离 -180° 还剩多少——这就是相位裕度，是判断电源稳定性的"体检报告"。**

![图6：波特图三步看稳定——增益穿0dB频率→读相位裕度→判余量够不够（含增益裕度）](/images/14/ill06_bode.png)

## 五、怎么测？频域注入扰动，时域看振铃

这条"环路增益曲线"，工程师怎么测出来？两条路：

**1. 频域直接测（最权威）——频响分析仪 / 网络分析仪。**
在反馈环路里串入一个小电阻，用**频响分析仪（FRA）**往里注入一个扫频的小扰动信号，测出环路在各个频率下的**增益和相位**，直接画出波特图，读相位裕度和增益裕度。这是评估环路稳定性的**标准方法**，结果最直接。

**2. 时域间接看（快速判断）——示波器看阶跃响应。**
没有频响分析仪时，可以用上一篇的**负载阶跃**法：给一个负载突变，看输出电压**回稳的样子**——
- **利落地一下回稳、不过冲** → 相位裕度足（大概 ≥ 60°）；
- **回稳时上下振铃几下才停** → 相位裕度不足（振得越多、越久，裕度越小）；
- **持续等幅 / 发散振荡** → 相位裕度 ≤ 0°，已自激。

所以上一篇那个"回稳不能振铃"的判据，本质就是在用时域**间接估相位裕度**。振铃，就是稳定性在向你报警。

> **一句话记住：测环路稳定性，频域用频响分析仪注入扰动、直接画波特图读相位裕度（最权威）；手边没有就用示波器看负载阶跃的振铃——回稳越干脆裕度越足，振铃越凶裕度越小。**

![图7：两种测法——频响分析仪注入扰动画波特图(频域) / 示波器看阶跃响应振铃反推(时域)](/images/14/ill07_how_to_test.png)

## 六、电源振荡了，怎么救？

如果测出来相位裕度不够、甚至已经振荡，怎么把它"摁稳"？核心思路：**要么把相位在临界频率前补回来，要么把增益压下去，让它在相位到 -180° 之前就掉到 0dB 以下。** 常用几招：

1. **调补偿网络（最根本）**——稳压器的反馈环路上有一组补偿元件（电阻电容），通过它在合适的频率加一个**零点**，把相位"托住"、把裕度补回来。这就是电源设计里说的 **Type II / Type III 补偿**（细节是专门的环路补偿设计，这里给方向）。
2. **调输出电容与 ESR**——输出电容的容值和**等效串联电阻（ESR）**会显著影响环路的相位。有时候"换一颗 ESR 合适的电容"就能把裕度救回来；反过来，上一篇为了瞬态盲目堆低 ESR 大电容，可能反而把环路推向不稳。**这就是瞬态和稳定性的"相爱相杀"。**
3. **降低环路带宽**——把增益穿越频率往低挪，让它在相位还没滞后到危险区之前就跌破 0dB。**代价是瞬态响应变慢**（电压坑变深）——又回到了上一篇那个"快与稳"的权衡。

看到没？**第13篇要"快"（提带宽）、第14篇要"稳"（够裕度），这俩天生打架。** 真正的功夫，是在两者之间找那个**既够快、又够稳**的平衡点——这就是电源环路设计的精髓。

> **一句话记住：救振荡三招——调补偿网络加零点托住相位（最根本）、调输出电容/ESR、降带宽（牺牲瞬态）。而瞬态要快、环路要稳，本就是一对矛盾，环路设计就是给它们找平衡点。**

## 七、一页速查：环路稳定性与相位裕度要点

| 关键词 | 一句话本质 | 新手常踩的坑 |
|--------|-----------|-------------|
| 自激振荡 | 反馈环路失稳，自己变振荡器 | 以为是某个元件坏了 |
| 负反馈→正反馈 | 延迟到半周期，纠偏变助推 | 以为负反馈一定越来越稳 |
| -180° 生死线 | 相位滞后 180°＋增益≥1 就振荡 | 只看增益、忘了看相位 |
| 相位裕度 PM | 增益穿 0dB 处相位离 -180° 的余量 | 不留余量、卡着临界设计 |
| >45° / <45° / ≤0° | 安全 / 振铃 / 自激 | 以为"不振荡"就够了 |
| 增益裕度 GM | 相位 -180° 处增益低于 0dB 的量 | 只看相位裕度、漏看增益裕度 |
| 波特图 | 增益+相位两条曲线判稳定 | 不会找穿越频率读裕度 |
| 振铃 = 报警 | 时域振铃＝相位裕度不足 | 把振铃当正常、不当回事 |
| 快与稳矛盾 | 提带宽要快、够裕度要稳 | 盲目提带宽/堆电容把环路搞不稳 |

## 写在最后

我们从那个"好端端却自己抖起来、还吱吱啸叫"的谜题出发，现在可以收网了：

**稳压器靠负反馈稳压，但反馈有延迟；当某个频率上延迟达到半个周期、也就是相位滞后 180°，而那里的增益还大于 1，负反馈就翻转成正反馈——纠偏变助推，系统自己振荡起来，就像推秋千踩错了点、话筒凑近音箱啸叫。** 而**相位裕度**，就是衡量"离这条 -180° 生死线还有多远"的安全余量：大于 45° 才稳，不到就振铃，到 0° 就自激。

> **环路稳定性第一准则：电源不是"反应越快越好"，而是要在"够快"和"够稳"之间留足相位裕度。看一张波特图——增益穿 0dB 处相位离 -180° 还剩多少——就知道这电源离自己变成振荡器，还有多远。**

### 判据表 1 · 环路稳定性核心判据

| 指标 | 含义 | 判据 |
|------|------|------|
| 相位裕度 PM | 增益穿越频率(增益=0dB)处，相位与 -180° 的差 | **≥45° 安全，目标常取 45°–60°**；<45° 振铃；≤0° 自激 |
| 增益裕度 GM | 相位穿越频率(相位=-180°)处，增益低于 0dB 的量 | **≥6dB（常要 ≥10dB）** |
| 增益穿越频率 fc | 开环增益降到 0dB(=1) 的频率，≈ 环路带宽 | 决定瞬态快慢；越高越快但越难稳 |
| 振荡条件 | 相位=-180° 且 增益≥1 同时满足 | 任一不满足则不自激 |

### 相位裕度 ↔ 时域过冲（经验对应，二阶近似）
| 相位裕度 | 阶跃过冲 | 表现 |
|------|------|------|
| 76° | ≈ 0% | 无过冲、最稳（偏慢）|
| 60° | ≈ 10% | 常用目标，干脆 |
| 45° | ≈ 23% | 临界可用，有振铃 |
| 30° | ≈ 37% | 明显振铃 |
| ≤0° | 发散 | 自激振荡 |

### 波特图读法（三步）
1. 上图找**增益穿过 0dB** 的频率 fc；
2. 下图在 fc 处读相位，**相位裕度 = 相位 −(−180°) = 180° + 相位**；
3. 找相位 = −180° 的频率，看上图增益低于 0dB 多少 = **增益裕度**。

### 测试方法
- **频域（标准）**：环路串入注入电阻 → **频响分析仪 FRA / 带 FRA 的示波器 / 网络分析仪**注入扫频小信号 → 测开环增益与相位 → 画波特图读 PM/GM。
- **时域（快速）**：示波器看**负载阶跃**响应——过冲/振铃量反推相位裕度（利落≈≥60°，振铃≈30–45°，发散≤0°）。
- 注意：注入点要选在反馈分压处；小信号幅度不能过大（保持线性）。

### 改善 / 补偿手段
- **补偿网络**：Type I / II / III（在反馈环加极点零点托相位）；电流模 vs 电压模补偿策略不同。
- **输出电容与 ESR**：ESR 引入零点影响相位；陶瓷电容(低 ESR)与电解/钽电容(高 ESR)对环路影响不同——换电容可能救稳定也可能破坏稳定。
- **降带宽**：fc 往低挪，牺牲瞬态换裕度。
- 经验：目标 PM 45°–60°、GM ≥10dB，兼顾瞬态与稳定。

### 踩坑 checklist
- [ ] 别只看"有没有振荡"——没振荡但裕度不足(振铃)也是隐患。
- [ ] 同时看相位裕度**和**增益裕度，别漏一个。
- [ ] 为瞬态盲目提带宽 / 堆低 ESR 大电容，可能把环路推向不稳。
- [ ] 注入点、注入幅度要对，否则波特图测不准。
- [ ] 温度/负载/输入电压变化会影响裕度，需在极端工况复测。
