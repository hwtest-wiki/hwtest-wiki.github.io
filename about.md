# 关于本站

**硬件测试科普百科**——把消费电子「硬件测试」的知识，做成一套可长期查阅、可全文搜索的完整百科。

每篇文章分两层：

- **科普层**：用大白话 + 类比 + 真实案例，把一个测试项讲透，零基础也能读懂。
- **工程师速查层**（🔧 标记）：判据表 / 规格值 / 设备设置（含 SCPI）/ 踩坑 checklist / 标准引用，方便工程师随手查。

## 模块导航

- **模块一 · 测试基础与方法论**——示波器、探头接地、工具箱、测量与校准
- **模块二 · 电源完整性 PI**——纹波噪声、压降 PDN、去耦、IR drop、瞬态响应、环路稳定性…
- **整机防护与可靠性**——IP 防水等级…

> 后续会持续补全信号完整性 SI、时序、总线接口、功耗电池、EMC/ESD、热设计、无线射频、整机可靠性等模块。
