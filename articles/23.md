# 输入才 5V，开关节点却冲到 12V——MOS 被反复打穿、EMC 也过不了

<div class="article-meta"><span class="badge">📘 模块二 · 电源完整性 PI · 开关节点振铃与过冲 · DCDC 专属</span><span class="badge">⏱ 约 15 分钟</span><span class="badge qr">🔧 含工程师速查</span></div>


先说个特别隐蔽、又特别烧钱的现场。

一款 DCDC 降压电源，输入**就 5V**。板子功能全正常，可产线上却**批量出现 MOS 管无故击穿**；送去做 EMC 认证，**辐射发射还超标、过不了**。

工程师上示波器，把探头点到 DCDC 的**开关节点**上一看——好家伙：每次开关的那一瞬间，电压都会窜起一个**尖尖的过冲**，直接冲到 **12V**，后面还拖着一长串**高频的振铃**（一种越来越小的来回抖动）。

这颗 MOS 的耐压（VDS）标着 20V，看着好像够。可你架不住这 **12V 的尖峰每个开关周期砸它一次**、一秒砸几十万次，再算上降额余量，它的"安全垫"早就被吃光了——**反复电压应力，把它慢慢打穿**。而那串几十上百兆赫兹的振铃，正是 **EMC 辐射超标的头号元凶**。

稳态波形好好的，全栽在**开关那一纳秒的过冲和振铃**上。今天这篇，就把这个 DCDC 里最"脏"的点讲清楚。

**读完这篇，你能搞懂：**

- ✅ **开关节点（SW node）是什么**——为什么它天生是个高速方波、是 DCDC 最"吵"的点；
- ✅ **过冲和振铃从哪来**——寄生电感和 Coss 组成的一场"共振"，被开关的电流突变激起来；
- ✅ 它会惹**两个祸**——把 MOS 反复电压应力打穿、当 EMI 辐射源让你过不了认证；
- ✅ 这段过冲振铃**怎么测**——为什么要高带宽 + 短地弹簧，长地线会测出"假振铃"；
- ✅ 怎么**压下去**——收紧回路、加缓冲、放慢边沿，各自的代价是什么。

## 一、开关节点是什么：DCDC 里那个"荡秋千"的点

先认识主角。DCDC 降压（buck）里有两个轮流开关的 MOS：**上管**和**下管**。它俩中间那个点，接着储能电感，就是**开关节点（SW node，也叫相位节点 / LX）**。

它是这么工作的：**上管开**的时候，SW 被拉到接近输入电压（≈5V）；**下管开**的时候，SW 被拉到接近地（≈0V）。上下管一秒钟轮流开关几十万到几百万次，于是 SW node 就成了一个**在输入电压和地之间高速来回切换的方波**，频率就是**开关频率**（几百 kHz 到几 MHz）。

这个"荡秋千"的动作，正是 DCDC 把电压降下来的原理。但也正因为它**电压跳变最快、电流变化最猛**（dv/dt、di/dt 都是全电路最大的），它天生就是 DCDC 里**最"吵"、最"脏"的一个点**。

理想情况下，SW 应该是一个干净利落的方波。可现实里，你把示波器点上去会发现——**每个方波的边沿上，都顶着一个过冲尖峰，后面还拖着一串振铃**。问题就出在这些边沿上。

> **一句话记住：开关节点 SW node = buck 里上、下管中间接电感的那个点，被上下管轮流拉到输入电压和地之间，形成一个开关频率的高速方波。它是 DCDC 里 dv/dt、di/dt 最大、最"吵"的点，过冲和振铃就长在它的每个边沿上。**

![图2：开关节点是什么——buck 拓扑上管/下管中间接电感的点，上管开时 SW≈输入电压、下管开时 SW≈0，形成开关频率的高速方波](/images/23/ill02_swnode.png)

## 二、过冲和振铃从哪来：寄生电感和 Coss 的一场共振

那边沿上的过冲和振铃，到底哪来的？一句话：**寄生电感和寄生电容的一场共振，被开关瞬间的电流突变点着了。**

拆开看：

- **开关切换极快**（几个纳秒），电流方向在这一瞬间几乎是"急刹车再反向"，**电流变化率（di/dt）巨大**。
- **回路里到处是寄生电感**：PCB 走线、器件封装的引脚、电容自身的等效电感（ESL）……这些加起来就是**功率回路电感（loop inductance）**。而电感上的电压等于电感值乘以电流变化率——巨大的 di/dt 撞在这个寄生电感上，就**激起一个电压尖峰，这就是过冲**。
- **这个寄生电感，又和 MOS 的输出电容 Coss（以及其它寄生电容）凑成了一个 LC 谐振槽**。开关这一下把它一激励，它就以自己的**固有频率来回振荡**——这就是**振铃**，频率常在**几十到几百兆赫兹**。

打个比方，这就像水管里的**水锤效应**：水（电流）流得好好的，你猛地关阀门（开关急停），水流的惯性撞上管壁，"哐"地一声顶起压力尖峰，还来回哐当几下才平息。电流的"惯性"就存在寄生电感里，撞上寄生电容，来回倒腾，衰减振荡。

所以记住这个因果：**功率回路电感越大、开关速度越快，过冲就越高、振铃就越凶。** 这也是后面所有"怎么压"的出发点。

> **一句话记住：过冲 = 开关瞬间巨大的电流变化率（di/dt）撞在功率回路的寄生电感上，激起的电压尖峰；振铃 = 这个寄生电感和 MOS 输出电容 Coss 组成 LC 谐振槽，被激励后以几十~几百 MHz 固有频率的衰减振荡。回路电感越大、开关越快，越严重。**

![图3动图：放大开关节点的一个上升沿——电压先冲过输入电压形成过冲尖峰，再以固有频率来回振荡、逐渐衰减到稳态，这就是振铃](/images/23/ill03b_ring_anim.gif)

## 三、它会惹两个祸：打穿 MOS，还让你过不了 EMC

这点过冲振铃，稳态时看不见，却实实在在地惹两个大祸。

**祸一：把 MOS 反复电压应力打穿。**

那个过冲尖峰，是**叠加在输入电压上**的，可能远超你以为的电压——输入 5V，SW 尖峰却能到 10V、12V 甚至更高。MOS 的耐压（VDS 额定）本来要留**降额余量**（比如只用到额定的 80%），可这尖峰**每个开关周期砸它一次、一秒几十万次**。轻则吃光余量、加速老化，重则直接击穿——就是开头那台"批量烧管"的现场。它不是一次烧的，是**被反复应力慢慢磨死的**，特别隐蔽。

**祸二：当 EMI 辐射源，让你 EMC 认证过不了。**

那串几十到几百兆赫兹的振铃，能量集中在高频，是 DCDC **最大的辐射发射源**。它顺着走线、器件当天线往外辐射，直接让你的 **EMC 辐射测试超标、过不了认证**（这是能不能上市的硬门槛，我们模块八会专门讲）。它还会**耦合进旁边的敏感电路**，变成噪声——射频灵敏度下降、音频有底噪，都可能是它干的（呼应第 8 篇的纹波与噪声）。

> **一句话记住：过冲振铃惹两个祸——① MOS 电压应力：过冲尖峰叠在输入上、每周期反复砸，吃光 VDS 降额余量甚至击穿（隐蔽的批量早期失效）；② EMI 辐射源：几十~几百 MHz 的振铃是 DCDC 最大辐射源，让 EMC 超标、还耦合成邻近电路噪声。**

![图4：两大危害——① 过冲尖峰叠在输入电压上、每个开关周期反复冲击 MOS，吃掉 VDS 降额余量导致反复击穿；② 高频振铃成为 EMI 辐射源，让 EMC 辐射测试超标](/images/23/ill04_harm.png)

![图4动图：功率回路电感越大 / 开关越快，过冲尖峰越高，一步步逼近并冲破 MOS 的 VDS 额定线，降额余量被吃光](/images/23/ill04b_stress_anim.gif)

## 四、怎么测：高带宽 + 短地弹簧，别用长地线

开关节点这段过冲振铃，是**全电路最考验测法**的信号之一——测不对，你要么看不到，要么看到一堆**假的**。两件事必须做对：

**① 带宽要够。** 振铃在几十到几百兆赫兹，示波器和探头的带宽至少要 **500MHz ~ 1GHz**。带宽不够，等于给信号加了个低通滤波器，把尖峰**削平了**——你测出来的过冲比真实的小得多，误以为"没问题"。

**② 探头地线：一定用短地弹簧，别用长地线！** 这是这篇最大的坑。

如果你图省事，用探头自带的那根**鳄鱼夹长地线**去量 SW node——**那根长地线本身就是个大电感**，它和探头的输入电容又凑成一个谐振回路，会**测出一大串又大又假的振铃**。这串振铃**不是电路的，是你的测法凭空造出来的**，会把你带沟里。

正确做法：用**短地弹簧（ground spring）**，把探头笔尖直接点在 SW node、地弹簧就近扎在**最近的功率地**上，让地回路的环路面积**尽可能小**。这样量到的，才是电路真实的过冲和振铃。

**探哪、读什么：** 探头点在 SW node 对最近的功率地，看每个开关边沿。读三个数：① **过冲峰值**——和 MOS 的 VDS 额定比，看降额余量还够不够；② **振铃频率**——用来反推是哪个 LC 在响、好对症下药；③ **是否落在 EMI 关注频段**——判断它会不会捅 EMC 的篓子。

> **一句话记住：测 SW node 过冲振铃 = 高带宽（示波器/探头 ≥500MHz~1GHz，否则削平尖峰）＋ 短地弹簧就近接最近功率地（长地线自身电感会造出"假振铃"，害人）。读三个数：过冲峰值 vs VDS 额定、振铃频率、是否落在 EMI 频段。**

![图5：正确测法——探头笔尖点 SW node、短地弹簧就近接最近功率地，环路面积最小；对比用鳄鱼夹长地线，长地线电感会测出又大又假的振铃](/images/23/ill05_probe.png)

![图5动图：同一个开关节点，长地线量出来是一大串又大又假的振铃、短地弹簧量出来是真实的小过冲——差别全在探头地线，别被长地线骗了](/images/23/ill05b_probe_anim.gif)

## 五、怎么压：先收紧回路，再谈阻尼和放慢边沿

知道了"回路电感越大、开关越快越严重"，压制的思路就很清楚了，**按性价比从高到低**：

- **先收紧功率回路（最该先做、最有效）**：把输入端的**高频陶瓷电容尽量贴近上下管**，让"上管—下管—输入电容"这个**开关回路的面积最小**。回路面积小，寄生电感就小，过冲和振铃从根上就小了。这是**布局（layout）**的活，也是免费的、最优先的一招。
- **加缓冲电路（snubber）**：在下管两端并一个 **RC 缓冲（snubber）**，给振铃提供**阻尼**，把振荡能量**耗散掉**。能有效压振铃，代价是 snubber 电阻上会有一点额外损耗。
- **放慢开关边沿**：适当**加大栅极电阻 / boot 电阻**，减小开关的电流变化率（di/dt），过冲振铃就轻。但**代价是开关损耗上升、效率下降**（呼应第 15 篇效率）——这是个**权衡**，不能一味放慢。
- **选器件**：必要时选 **Coss 更合适、封装寄生更小**的 MOS，或用集成度高、内部回路更紧的电源模块。

一句话总结优先级：**能靠布局收紧回路解决的，绝不先上 snubber；能小改的，绝不牺牲效率去放慢边沿。**

> **一句话记住：压过冲振铃的优先级——① 收紧功率回路（输入电容贴近上下管、回路面积最小，最有效且免费）；② 下管加 RC 缓冲 snubber 提供阻尼；③ 加大栅极/boot 电阻放慢边沿（有效但牺牲效率，需权衡）；④ 选低寄生器件/电源模块。**

![图6：三招压制过冲振铃——① 收紧功率回路（输入电容贴近上下管、回路面积最小）② 下管加 RC snubber 提供阻尼 ③ 加大栅极/boot 电阻放慢边沿（牺牲效率），按性价比从上到下](/images/23/ill06_fix.png)

## 七、写在最后

这一篇想让你记住一件容易被忽略的事：**DCDC 的稳态波形再干净，都不代表它安全。** 真正的杀机，藏在**开关那一纳秒**——开关节点上那个冲过头的过冲尖峰、和后面那串高频振铃。

它稳态时你看不见，却在**反复地磨着 MOS 的耐压余量**，也在**源源不断地往外辐射**。前者让你产线上莫名其妙地批量烧管，后者让你 EMC 认证卡关。而这一切，只有你上**高带宽示波器、用短地弹簧**贴着开关节点量一次，才看得见真相——**千万别用长地线，那会给你造一串假振铃，把你带沟里。**

下次再碰到"MOS 无故击穿 / EMC 辐射超标 / 开关噪声大"，别急着换管子——先量量你的开关节点，看看那个过冲有多高、振铃有多凶，再从**收紧回路**开始下手。

下一篇，我们顺着"高频噪声"这条线往下走——**用 FFT 给纹波和噪声溯源**：示波器上看着一团乱的噪声，怎么用频谱把它拆开，一眼看出到底是开关频率、是这串振铃、还是别的谁在捣乱。

### 判据表 1 · 开关节点振铃与过冲核心判据

| 项 | 测什么 | 判据 / 参考 |
|---|---|---|
| **过冲峰值 V_SW,pk** | 开关边沿最高电压（叠在输入上） | ≤ MOS VDS 额定 × 降额系数（如 ≤80%），留够余量 |
| **振铃频率 f_ring** | 衰减振荡的频率 | 定位对应 LC（Lloop×Coss）；用于对症下药 |
| **振铃衰减/持续** | 振荡几个周期内衰减 | 越快越好；持续久＝阻尼不足 |
| **测量带宽** | 示波器 + 探头带宽 | ≥ 500MHz~1GHz，否则削平尖峰失真 |
| **探头地线** | 接地方式 | **短地弹簧**就近接最近功率地；禁用长地线（造假振铃） |
| **EMI 频段** | 振铃频率是否落在关注频段 | 落在 EMC 辐射频段＝高风险，需压制 |
| **功率回路布局** | 输入电容到上下管的回路面积 | 越小越好，回路电感最小化（第一优先） |

### 关键概念与公式

- **开关节点 SW node（相位节点/LX）**：buck 上下管中间接电感的点，在输入电压与地之间高速切换的方波
- **过冲 overshoot**：开关瞬间 di/dt 撞在功率回路寄生电感 Lloop 上激起的电压尖峰，**V ≈ L·di/dt**
- **振铃 ringing**：Lloop 与 MOS 输出电容 Coss 组成 LC 谐振槽的衰减振荡，**f_ring ≈ 1 / (2π√(Lloop·Coss))**，常几十~几百 MHz
- **功率回路电感 Lloop**：输入电容—上管—下管开关回路的寄生电感（走线+封装+ESL）；越小越好
- **VDS 额定与降额**：MOS 漏源耐压；过冲峰值须落在额定的降额余量内（如 ≤80%）
- **越严重的两个因素**：Lloop 越大、开关 di/dt 越快（边沿越陡）

### 测试设置

- **仪器**：高带宽示波器（≥500MHz~1GHz）、匹配带宽的无源/有源探头
- **探点**：探头笔尖点 SW node，**短地弹簧**就近接**最近的功率地**，环路面积最小
- **禁忌**：**不要用鳄鱼夹长地线**——长地线电感会与探头电容谐振，造出又大又假的振铃
- **触发**：正常/单次触发，展开单个开关边沿看过冲峰值与振铃
- **读数**：① 过冲峰值 vs VDS 额定（降额）② 振铃频率（定位 LC）③ 是否落在 EMI 关注频段
- **交叉**：配合近场探头 / 频谱仪看该频点的辐射（EMC 排查）

### 常见翻车与对策

- **MOS 批量无故击穿** → 过冲超 VDS 余量：收紧回路降 Lloop、加 snubber、选更高耐压/低 Coss 器件
- **EMC 辐射某高频超标** → 振铃在辐射：先测振铃频率与超标频点是否吻合，再收紧回路+加阻尼
- **量出一大串巨大振铃** → 多半是长地线假振铃：换短地弹簧就近接地重测，别被骗
- **过冲测出来偏小、却还烧管** → 带宽不够削平了尖峰：换更高带宽示波器/探头
- **加 snubber / 大栅极电阻后效率掉了** → 阻尼与放慢边沿都牺牲效率：优先靠布局收紧回路，阻尼取够用即可
- **换了管子还是烧** → 没解决根因（回路电感/过冲）：从 layout 和过冲峰值入手，而非只换器件
