# 纹波超标滤了一下午只降 4mV——FFT 一看，我全滤错了地方

<div class="article-meta"><span class="badge">📘 模块二 · 电源完整性 PI · 用 FFT 给纹波和噪声溯源 · 通用</span><span class="badge">⏱ 约 19 分钟</span><span class="badge qr">🔧 含工程师速查</span></div>


一块板子，输出纹波噪声测出来 **30mV**，可 datasheet 要求 **≤20mV**——超了。

你二话不说，先加大输出电容，又补了一级 LC 滤波，折腾一下午，回头再测——**只从 30mV 降到 26mV**，还是过不了。你开始怀疑人生：电容都堆成这样了，怎么还压不下去？

后来你把那段波形做了一次 **FFT**，屏幕上蹦出一张频谱图，真相一下就清楚了：顶在超标线上的那根尖峰，**根本不在你的开关频率上**（那根你一下午都在滤，它其实早就达标了）——而是在 **2.1MHz** 上孤零零冒出来的一根线。查一下，那正好是**隔壁另一颗 DCDC 的开关频率**，顺着地和走线耦合过来的。

你一下午加的那些电容，全加错了地方。

这就是时域的天花板：它只告诉你"噪声有 30mV"，却不告诉你**这 30mV 是谁凑出来的**。而 FFT，就是把这团糊在一起的噪声，**按频率一根根拆开、对号入座**的那把尺子。

**读完这篇，你能搞懂：**

- ✅ 时域波形的**天花板**——为什么它只能告诉你"多大"，看不出"是谁"；
- ✅ **FFT 到底在干什么**——怎么把一段波形拆成一根根"频率身份证"；
- ✅ 频谱上每根线**分别是谁**——开关基频、谐波、市电低频、拍频、展频、高频振铃，逐个点名；
- ✅ 示波器 FFT **怎么设才不骗你**——窗函数、频率分辨率、混叠，设错了它也会造假线；
- ✅ 从谱线**到改哪里**——低频、高频、拍频各自的治法完全不同，别再对着时域瞎加电容。

## 一、时域的天花板：你只看得见"多大"，看不见"谁"

先说清楚，时域波形（横轴是时间那张图）不是没用——第 8 篇整篇都在教你怎么在时域把纹波噪声**测准**。它的问题是：**它只给你一个总量。**

你在示波器上量到一个 30mV 的 Vpp，这 30mV 其实是**好几路东西叠在一起**的结果：开关电源自己的规律纹波、开关瞬间的高频振铃、隔壁电路耦合过来的干扰、市电那点低频波动……它们全骑在这一条线上，晃成一团。时域只会把它们**加起来**告诉你"一共 30mV"，至于**每一路各占多少、分别从哪来**——它一个字都不说。

打个比方：这就像你站在一个吵闹的房间外面，只听得见里头"很吵"，却分不清是几个人在说话、各自在说什么。你想让它安静，可你连**谁在大声**都不知道，怎么去劝？

**FFT 干的就是"声纹分离"这件事**——把这团混在一起的声音，按每个人的音调（频率）拆开，让你一眼看出：哦，最吵的是这位（开关频率），旁边还有一个一直在小声嘀咕的（耦合干扰）。

> **一句话记住：时域波形只给你一个"总量"（Vpp），它把开关纹波、高频振铃、耦合干扰、低频波动全加在一起，却不告诉你各自占多少、从哪来。想追源头，光看时域到头了——得换个维度，用 FFT 看频率。**

## 二、FFT 是什么：把一段波形拆成一根根"频率身份证"

FFT，全称快速傅里叶变换。名字唬人，道理其实一句话就能说清：**任何一段波形，都可以看成一堆不同频率的正弦波叠加起来的结果。** FFT 就是把这个叠加**倒着算回去**——告诉你这段波形里，到底掺了哪些频率、每个频率有多强。

它把一张**时域图**（横轴时间、纵轴电压）换算成一张**频域图**（横轴频率、纵轴幅度），我们叫它**频谱**。换完之后：

- 一个**周期性、有规律**的成分（比如开关纹波），会在它对应的频率上，立起**一根又高又尖的线**；
- 一堆**杂乱无规律**的随机噪声，会在频谱上摊成一层**矮矮的底噪**，没有明显的尖峰；
- 一个**单一频率的干扰**（比如别处耦合来的），会在那个频率上，孤零零冒出**一根线**。

关键就在这里：**频率，就是每根线的身份证。** 一根线立在 500kHz，它八成就是那颗开关频率 500kHz 的 DCDC；一根线立在 2.1MHz，你就去查哪颗电路工作在 2.1MHz。时域里糊成一团分不清的东西，到了频域，**按频率一站队，各自的来历就藏不住了**。

这等于给你的纹波噪声拍了一张"**频率 CT**"——哪块有问题、问题在哪个频段，一目了然。

> **一句话记住：FFT 把时域波形（横轴时间）换算成频谱（横轴频率）。周期性成分在对应频率立一根尖线、随机噪声摊成底噪、单一干扰冒一根孤立线。频率就是每根线的"身份证"——时域分不清的，频域里按频率一站队就现形。**

## 三、对号入座：频谱上每根线，到底是谁

频谱拉出来了，接下来才是真功夫：**这一根根线，分别是谁？** 电源噪声的频谱，常见的就这么几类"住户"，认全了，你就能对号入座。

**① 开关基频（最高那根，纹波主体）。** DCDC 以某个固定频率一开一关地搬电，这个频率（常见几百 kHz 到几 MHz）上会立起**最高的一根线**——这就是纹波的主体。它多大，基本决定了你规律纹波的大小。

**② 谐波（基频的整数倍）。** 基频旁边，还会在它的 2 倍、3 倍、4 倍频上，立起一排**逐渐变矮**的线，这叫谐波。开关波形的**边沿越陡**（呼应第 23 篇的开关节点），谐波就越丰富、拖得越高频。

**③ 低频线（100Hz / 120Hz 及其倍数）。** 频谱最左边如果有低频线，多半来自**前级的 AC-DC**：市电 50Hz（或 60Hz）整流之后是 100Hz（或 120Hz），这点工频纹波会一路渗到后级。看到它，就往电源输入端、bulk 电容那边查。

**④ 拍频 / 差频（最迷惑人的那根）。** 如果板上有**两颗开关频率很接近**的 DCDC（比如一颗 2.0MHz、一颗 2.1MHz），它俩会"打架"，在**两者之差**的频率上（这里就是 0.1MHz = 100kHz）凭空拍出**一根新的低频线**。它不在任何一颗芯片的标称频率上，最容易让人摸不着头脑——**开头那个翻车现场，顶在超标线上的就是它**。

**⑤ 展频摊平的包（spread spectrum）。** 有些 DCDC 会开**展频**功能：故意让开关频率在一个小范围里来回抖。这样一来，原本又高又尖的开关基频那根线，就被**摊平成一个矮矮的包**——峰值降下来了（为的是过 EMC），但它盖住的**总能量并没有变**。你在频谱上看到"一个鼓包而不是一根尖线"，多半就是它开了展频。

**⑥ 高频振铃（几十~几百 MHz 的一簇）。** 频谱很高频的地方，如果有一簇鼓起来的东西，那多半是开关节点的**振铃**（第 23 篇的主角）。它是 EMI 辐射的头号来源，也会耦合成邻近电路的噪声。

把这六类认熟，你拿到任何一张电源频谱，基本都能**指着每根线说出它是谁**。

> **一句话记住：电源频谱常见六类线——① 开关基频（最高、纹波主体）② 谐波（基频整数倍、边沿越陡越多）③ 低频 100/120Hz（前级市电整流渗过来）④ 拍频（两颗频率相近的 DCDC 拍出的差频，最迷惑人）⑤ 展频摊平的矮包（峰值降但总能量不变）⑥ 高频振铃簇（几十~几百 MHz，EMI 源）。认出频率，就认出了源头。**

## 四、示波器 FFT 别乱设：设不对，它也会骗你

这里要泼盆冷水：**FFT 不是按一下就出真相的魔法。** 示波器上做 FFT，有几个设置你不懂，它照样给你一张"看着专业、其实骗人"的频谱。三件事必须心里有数：

**① 频率分辨率（RBW）：能不能把两根近邻的线分开。** 你抓的这段波形**时间越长**，FFT 出来的频率分辨率就**越细**，越能把挨得很近的两根线分开。反过来，窗口太短，分辨率粗，第三节那个"2.0MHz 和 2.1MHz 拍频"的两根线就会**糊成一根**，你根本看不出是两颗在打架。想看清细节，就得抓够长的一段。

**② 采样率不够 → 混叠（aliasing），凭空造假线。** 示波器采样率决定它最高能"看到"多高的频率（奈奎斯特：最高只能看到采样率的一半）。要是有个真实的高频成分超过了这条线，它不会消失，而会像车轮在视频里**倒转**那样，被**折射到一个错误的低频位置**上，冒出一根**根本不存在的假线**。你要是信了它去查电路，就白忙活。**看高频，先把采样率开够。**

**③ 窗函数（Window）：选错了，一根线糊成一片。** FFT 要求你截取的那段波形"首尾能接上"，可现实里截出来总是断的，这个断口会让频谱**泄漏**——一根本该干净的尖线，糊成一坨、还把旁边的小线盖住。加**窗函数**就是为了把这个断口"抹平"。常用的记住两个就够：

- **Hanning（汉宁窗）**：最通用，分辨频率和看幅度都还行，日常溯源默认用它；
- **Flattop（平顶窗）**：**测某根线的幅度最准**（想知道那根尖峰到底多少 dB 就用它），但频率分辨率差一些。

（还有矩形窗，分辨率最高但泄漏最重，一般不用来测幅度。）

至于**示波器 FFT 和专门的频谱分析仪怎么选**：示波器 FFT 方便、随手就能定位"是哪个频段在顶"，日常排查足够了；但它的**动态范围和本底噪声**远不如频谱分析仪——要看很微弱的谱线、或者认真量高频辐射，还得上频谱分析仪（这也是频谱仪第一次在本系列登场，EMC 篇会再用到它）。

> **一句话记住：示波器 FFT 有三个"骗人"的坑——① 抓的波形太短→频率分辨率粗，近邻两根线糊成一根（拍频看不出）；② 采样率不够→高频混叠，凭空折射出假线；③ 窗函数选错→频谱泄漏，尖线糊一片。日常溯源用 Hanning 窗、想量准某根线幅度用 Flattop 窗；要看微弱/高频信号，上频谱分析仪。**

![图4：示波器 FFT 的三个坑——① 时间窗越长频率分辨率越细，才分得开近邻的两根线；② 采样率不够会把高频"混叠"折射成假的低频线；③ 窗函数选错导致频谱泄漏，一根尖线糊成一坨](/images/24/ill04_fftsetup.png)

## 五、从谱线到改哪里：按频率对症下药

认出是谁，还只是一半；**关键是不同频率的线，治法完全不一样**——这也是为什么开头那位对着时域瞎加电容会白忙一下午。按频段对症：

**低频的开关基频 / 低次谐波顶超标** → 这是规律纹波主体。往**滤波**上使劲：加大或优化输出滤波电容、加一级 LC 后级滤波，或者在 DCDC 后面**再串一颗 LDO 做"二次清洗"**（呼应第 17 篇：LDO 低噪声）。加电容这招，**只对这一类真正管用**。

**高频的谐波 / 振铃簇** → 加电容基本没用（电容到高频早就"失效"了）。得从**源头和路径**下手：收紧功率回路、下管加 snubber、放慢开关边沿（第 23 篇那三招），再配合**去耦、屏蔽、改布局**。

**拍频 / 差频那根** → 这根最"轴"，**光加电容永远压不掉**，因为它不是滤出来的、是两颗 DCDC 频率打架拍出来的。正解是**让它们别打架**：把多颗 DCDC 的时钟**同步**起来（clock sync / 锁相），或者干脆把它们的开关频率**拉开距离**，差频自然就没了。开头那位要是早看一眼频谱，就知道该去做时钟同步，而不是加电容。

**100/120Hz 低频线** → 往**前级**查：AC-DC 的整流滤波、加大 bulk（储能）电容，把工频纹波在源头压住。

**展频摊平的包** → 它本身就是**用来压 EMC 峰值**的手段，不用"治"。但要留个心：展频降的是**峰值不是有效值**，而且开关频率在抖，对**时钟特别敏感的电路**可能引入一点抖动——用不用、抖多宽，是个权衡。

看出门道了吗：**FFT 把"噪声很大"这句模糊的抱怨，翻译成了"某某频率在顶、来自某某"，你才谈得上对症。** 频率认错，方向就错，电容加成山也白搭。

> **一句话记住：按频率对症——① 开关基频/低次谐波顶：加大滤波/LC 后级/后串 LDO（加电容只对这类有效）；② 高频谐波/振铃：收紧回路+snubber+去耦屏蔽（加电容没用）；③ 拍频：做时钟同步或拉开开关频率（滤不掉）；④ 100/120Hz：查前级 AC-DC 滤波/bulk；⑤ 展频包：本就是压 EMC 峰值的手段、权衡时钟抖动。频率认错，方向就错。**

![图5：从频率到治法的对症地图——低频基频/谐波治滤波与后级 LDO、高频振铃治回路与去耦、拍频做时钟同步、100Hz 查前级、展频是压 EMC 峰值手段，每个频段对应各自的下手处](/images/24/ill05_fixmap.png)

## 七、写在最后

这一篇想让你记住一件事：**当纹波噪声超标、而你加了电容却压不下去时，先别较劲——换个维度看。**

时域波形是个老实但"嘴笨"的证人：它只会告诉你"一共 30mV"，却说不清这 30mV 是谁凑的。而 FFT，就是把它拆开审问的那把尺子——**开关基频、谐波、市电低频、拍频、展频、高频振铃，每一根线都在频率上写着自己的名字。** 认出频率，就认出了源头；认出源头，才知道该动滤波、动回路、还是动时钟。

回到开头那位：他一下午加的电容，方向从一开始就错了——顶在超标线上的是**另一颗 DCDC 拍出来的差频**，那是滤不掉的，得去做时钟同步。他要是先花一分钟做个 FFT，就不会白忙这半天。

所以下次再碰到"纹波噪声超标、越加电容越憋屈"，记住这个顺序：**先按第 8 篇测准，再做 FFT 看是谁在顶，最后按频率对症。** 别让一团看不清的波形，骗你把电容加错地方。

下一篇，我们顺着"电源噪声为什么会掉这么多"再往里走一层——讲 **PDN（电源分配网络）的目标阻抗**：为什么芯片一大口吃电，电压就会瞬间往下塌？怎么用一条**阻抗曲线**，在画板子之前就预判它会不会塌、该在哪个频段补多少电容。把"噪声"这件事，从"测和溯源"推进到"设计时就预防"。

|---|---|---|---|

### 判据表 2 · 示波器 FFT 设置要点

| 设置项 | 影响 | 怎么选 |
|---|---|---|
| **抓取时间窗长度** | 决定频率分辨率（RBW），能否分开近邻线 | 想分开拍频等近邻线 → 抓够长的一段 |
| **采样率 / 带宽** | 决定能看多高频；不够会混叠造假线 | 看高频 → 采样率开够（奈奎斯特 > 关注最高频 ×2） |
| **窗函数** | 抑制频谱泄漏 | 通用溯源 **Hanning**；量幅度准用 **Flattop**；矩形窗慎用 |
| **纵轴单位** | dBV（对数，看动态范围）/ 线性 | 找微弱谱线用 dB；直接比幅度用线性 |
| **平均（Average）** | 平滑随机底噪、凸显稳定谱线 | 多次平均让固定频率的线更清楚 |

### 关键概念与公式

- **FFT（快速傅里叶变换）**：把时域波形分解为各频率正弦分量，得到频谱（横轴频率、纵轴幅度）
- **频率 = 每根谱线的"身份证"**：谱线所在频率直接对应物理源（开关频率、市电、耦合源…）
- **谐波 harmonics**：基频的整数倍分量；波形边沿越陡（方波感越强）谐波越丰富
- **拍频 / 差频 beat**：两个相近频率 f1、f2 叠加，在 |f1 − f2| 处产生的新分量；**滤波无效，需错开或同步频率**
- **展频 spread spectrum**：主动抖动开关频率，把基频尖峰摊平成矮包，**降峰值不降总能量**，用于过 EMC
- **奈奎斯特 / 混叠 aliasing**：采样率必须 > 关注最高频率 ×2；否则高频折射成假的低频线
- **频率分辨率 RBW**：≈ 1 / 抓取时间窗长度；窗越长分辨率越细，越能分开近邻谱线
- **窗函数 window**：Hanning（通用）、Flattop（幅度准）、矩形（分辨率高但泄漏大）——抑制截断造成的频谱泄漏

### 测试设置

- **前置**：先按第 8 篇把时域测法对齐——AC 耦合、短地弹簧/tip-and-barrel、探到最靠近负载的去耦点；带宽按目的（比幅度对齐 datasheet 用 20MHz；追高频振铃/EMI 用全带宽）
- **仪器**：带 FFT 的数字示波器（日常溯源）；看微弱谱线 / 高频辐射用**频谱分析仪**
- **FFT 设置**：Hanning 窗（量幅度换 Flattop）、抓够长的时间窗提分辨率、采样率开够防混叠、必要时开平均
- **读谱**：先找最高那根（多半开关基频）→ 认谐波 → 扫低频（100/120Hz）→ 找不该出现的孤立线（拍频/耦合）→ 看高频簇（振铃）
- **复测**：改完用**同一套 FFT 设置**复测，对比谱线升降确认改对了

### 常见翻车与对策

- **加了一堆电容纹波纹丝不动** → 顶超标的多半不是开关基频：先 FFT 看是谁在顶，别对着总量瞎加
- **频谱冒出一根标称频率上没有的线** → 多半是两颗 DCDC 的拍频：做时钟同步或拉开频率，滤波无效
- **高频有根尖线，一查电路上没这频率** → 采样率不够导致的混叠假线：把采样率/带宽开够重看
- **两根近邻线看着像一根** → 时间窗太短、分辨率不够：抓更长一段重算
- **那根尖峰幅度每次读数都不一样** → 窗函数没选对（泄漏）：量幅度换 Flattop 窗
- **基频不是尖线是个矮包** → 开了展频：这是压 EMC 峰值的正常现象，别当故障；关注总能量与时钟抖动
- **示波器 FFT 看不到很弱的谱线** → 动态范围/本底噪声不够：上频谱分析仪
